# 速查表

## 决策规则

### 判断财政体制类型
| 特征 | 财政包干制（1980-1993） | 分税制（1994至今） |
|------|------------------------|-------------------|
| 收入划分 | 讨价还价确定分成比例 | 按税种划分中央税、地方税、共享税 |
| 中央收入占比 | ~30% | ~50-55% |
| 地方积极性 | 高（多收多得） | 中（依赖转移支付） |
| 主要问题 | 中央财力薄弱 | 基层财政困难 |
| 政企关系 | 放水养鱼、扶持乡镇企业 | 失去经营企业的热情 |

### 评估转移支付效果
| 指标 | 理想状态 | 实际状态 |
|------|----------|----------|
| 税收返还占比 | 低 | 高（维护既得利益） |
| 一般性转移支付占比 | 高 | 低 |
| 专项转移支付占比 | 适中 | 高（导致跑部钱进） |
| 均等化效果 | 显著 | 有限 |

### 判断基层财政健康状况
| 指标 | 健康 | 警示 | 危险 |
|------|------|------|------|
| 财政自给率 | >70% | 50-70% | <50% |
| 人员支出占比 | <60% | 60-80% | >80% |
| 转移支付依赖度 | <30% | 30-60% | >60% |
| 土地收入占比 | <30% | 30-50% | >50% |

## 决策树

### 如何分析地方政府行为
```
财政体制类型？
├── 财政包干制 → 观察包干基数和分成比例
│                 ├── 高分成比例 → 积极发展经济
│                 └── 低分成比例 → 藏富于企业
└── 分税制 → 观察转移支付依赖度
              ├── 低依赖 → 土地财政/预算外收入
              └── 高依赖 → 跑部钱进/资金沉淀
```

### 如何评估政策效果
```
政策目标明确？
├── 否 → 警惕模糊措辞，关注实际执行
└── 是 → 激励相容？
          ├── 是 → 政策可能有效
          └── 否 → 执行中会变形
                    ├── 倒逼 → 基层会反倒逼
                    └── 专项化 → 资金会沉淀
```

## 权衡矩阵

### 财政包干制 vs 分税制
| 维度 | 财政包干制 | 分税制 |
|------|-----------|--------|
| 中央控制力 | 弱 | 强 |
| 地方积极性 | 高 | 中 |
| 地区差距 | 扩大 | 理论上缩小 |
| 基层财政 | 相对宽裕 | 困难 |
| 政企关系 | 密切 | 疏远 |
| 市场统一 | 分割 | 相对统一 |

### 发达地区 vs 中西部地区
| 维度 | 发达地区 | 中西部地区 |
|------|----------|-----------|
| 主要财源 | 工商业税收、土地收入 | 转移支付 |
| 财政自给率 | 高 | 低 |
| 公共服务 | 相对较好 | 不足 |
| 土地财政 | 依赖度高 | 依赖度低 |
| 债务风险 | 高（融资平台） | 中（转移支付依赖） |

## 阈值与默认值

### 关键数据参考
- 分税制改革：1994年
- 税费改革：2002年开始试点，2006年全面取消农业税
- 增值税中央地方分成：75:25
- 中央财政收入占比（分税制后）：~50-55%
- 地方财政支出占比：~85%
- 融资平台债务（2009年末）：~6万亿
- 土地财政依赖度（发达地区）：~50-70%

### 改革关键节点
- 1980年：财政包干制开始推行
- 1994年：分税制改革
- 2002年：农村税费改革试点
- 2006年：全面取消农业税
- 2008年：四万亿经济刺激计划

## 快速识别信号

### 基层财政危机信号
- 财政自给率低于50%
- 人员支出占比超过80%
- 高度依赖转移支付
- 存在大量预算外收入

### 土地财政依赖信号
- 土地出让金占财政收入超过50%
- 征地补偿标准偏低
- 大规模城市扩张
- 融资平台债务快速增长

### 基层政权悬浮化信号
- 乡镇财政收入几乎全部来自转移支付
- 乡镇政府与农民缺乏直接联系
- 农村公共服务供给不足
- 乡镇财政空壳化
