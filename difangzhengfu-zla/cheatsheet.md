# 速查表

## 决策规则

### 判断行政发包制程度
| 条件 | 行政发包程度高 | 行政发包程度低 |
|------|---------------|---------------|
| 信息和监督成本 | 高 | 低 |
| 财政约束 | 紧 | 松 |
| 统治风险 | 低 | 高 |
| 典型领域 | 经济发展、招商引资 | 国防、外交、金融监管 |

### 判断政治锦标赛有效性
| 条件 | 有效 | 无效 |
|------|------|------|
| 考核指标 | 可量化、可比较 | 难以量化、难以比较 |
| 晋升竞争 | 激烈 | 不激烈 |
| 晋升期望 | 高 | 低（如年龄超限） |
| 外部约束 | 适度 | 过强或过弱 |

### 评估财政体制类型
| 特征 | 统收统支 | 财政包干制 | 分税制 |
|------|----------|-----------|--------|
| 时期 | 计划经济 | 1980-1993 | 1994至今 |
| 地方积极性 | 低 | 高 | 中 |
| 中央控制力 | 强 | 弱 | 强 |
| 预算外财政 | 少 | 多 | 较多 |
| 激励特征 | 弱激励 | 强激励 | 中等激励 |

## 决策树

### 如何分析地方政府行为
```
激励机制？
├── 政治锦标赛 → 观察考核指标和晋升竞争
│                 ├── GDP挂帅 → 追求经济增长
│                 ├── 环保考核 → 加强环境治理
│                 └── 民生考核 → 改善公共服务
└── 财政激励 → 观察财政收入和支出
              ├── 财政包干 → 积极创收
              ├── 分税制 → 依赖转移支付
              └── 土地财政 → 经营城市
```

### 如何评估区域合作可能性
```
晋升博弈性质？
├── 零和博弈 → 合作困难
│              ├── 利益补偿机制？
│              │   ├── 有 → 可能合作
│              │   └── 无 → 难以合作
│              └── 共同上级协调？
│                  ├── 有 → 可能合作
│                  └── 无 → 难以合作
└── 正和博弈 → 合作可能
```

## 权衡矩阵

### 行政发包制 vs 韦伯式官僚制
| 维度 | 行政发包制 | 韦伯式官僚制 |
|------|-----------|-------------|
| 激励方式 | 强激励（财政分成、晋升竞争） | 规则程序导向 |
| 灵活性 | 高 | 低 |
| 可预测性 | 低 | 高 |
| 腐败风险 | 高 | 低 |
| 短期行为 | 多 | 少 |
| 长期规划 | 弱 | 强 |
| 适用情境 | 快速发展、创新探索 | 稳定发展、规则执行 |

### 政治锦标赛 vs 其他激励机制
| 维度 | 政治锦标赛 | 公务员制度 | 选举制度 |
|------|-----------|-----------|----------|
| 激励强度 | 强 | 弱 | 中等 |
| 竞争对象 | 同级官员 | 无 | 其他候选人 |
| 考核标准 | 上级设定 | 规则设定 | 选民偏好 |
| 短期行为 | 多 | 少 | 中等 |
| 地方保护 | 强 | 弱 | 中等 |
| 适用情境 | 经济增长 | 公共服务 | 民主政治 |

## 阈值与默认值

### 关键数据参考
- 中国行政层级：中央—省（34）—市（约300）—县（约2800）—乡（约40000）
- 政治锦标赛晋升比例：逐级淘汰，越往上越难
- 干部异地交流：普遍实行，任期通常3-5年
- 分税制增值税分成：中央75%，地方25%
- 预算外财政占比：历史高峰期可达预算内50%以上

### 改革关键节点
- 1980年：财政包干制开始推行
- 1994年：分税制改革
- 1998年：国企改革攻坚
- 2002年：干部人事制度改革
- 2006年：取消农业税
- 2012年：十八大后淡化GDP挂帅
- 2018年：党和国家机构改革

## 快速识别信号

### 行政发包制特征信号
- 目标责任制和量化考核普遍
- 财政分成和预算包干
- 地方政府有较大自主权
- "一包就灵"的说法流行

### 政治锦标赛特征信号
- GDP增长目标层层加码
- 官员任期与投资周期同步
- 地方间激烈竞争
- 干部频繁异地交流

### 政府治理转型信号
- 垂直化管理加强
- 公共财政建设推进
- GDP考核权重下降
- 民众满意度调查引入
- 差额选举试点扩大
