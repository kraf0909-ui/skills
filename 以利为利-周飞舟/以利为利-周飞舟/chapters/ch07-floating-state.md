# 第七章：基层政权的"悬浮"状态

## Core Idea
2002年农村税费改革取消了农业税和"三提五统"，虽然减轻了农民负担，但导致乡镇财政收入锐减。乡镇政府几乎完全依赖上级转移支付，失去了与农民的直接联系，从"汲取型"转变为"悬浮型"政权。

## Frameworks Introduced
- **税费改革**：取消农业税和各项收费的农村改革
  - 当 to use: 分析农村基层政权变化、农民负担问题
  - How: 观察改革前后乡镇财政收入来源和结构变化
- **悬浮型政权**：基层政权脱离与民众的直接联系，悬浮于社会之上
  - 当 to use: 理解基层政府与民众关系的变化
  - How: 观察乡镇政府是否还直接面向农民提供服务
- **汲取型政权**：传统国家从社会中汲取资源的政权形态
  - 当 to use: 比较改革前后国家—农民关系
  - How: 观察政府是否直接从农民手中收取税费

## Key Concepts
- **农业税**：对农业收入征收的税收，2006年全面取消
- **三提五统**：村提留（公积金、公益金、管理费）和乡统筹（教育、计划生育、民兵训练、优抚、交通）
- **一事一议**：农村公益事业建设由村民民主商议决定的制度
- **转移支付依赖**：乡镇财政几乎完全依赖上级拨款

## Mental Models
- 用"悬浮"理解政权变化：乡镇政府不再直接接触农民，悬浮于社会之上
- 用"倒逼"理解改革逻辑：中央试图通过硬化预算约束倒逼基层精简机构

## Anti-patterns
- **反倒逼**：基层通过诉苦、变通等手段使中央的倒逼意图失败
- **空壳化**：乡镇财政有名无实，失去实际功能
- **公共服务缺位**：改革后农村公共服务反而恶化

## Key Takeaways
1. 税费改革减轻了农民负担，但造成乡镇财政危机
2. 乡镇政府从"汲取型"转变为"悬浮型"政权
3. 基层政府几乎完全依赖上级转移支付
4. 农村公共服务出现缺位现象
5. 中央与基层的"倒逼—反倒逼"博弈
