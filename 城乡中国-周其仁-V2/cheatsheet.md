# 速查手册（Cheatsheet）

## 核心论点速览

| 论点 | 核心内容 |
|------|----------|
| 城市是密度 | 城市的本质是经济密度，密度产生集聚效应 |
| 城市化滞后 | 中国城市化明显滞后于工业化和经济发展水平 |
| 自由是根基 | 城市起源于自由，自由是城市繁荣的制度基础 |
| 权利要界定 | 清晰的产权界定是市场交易和效率提升的前提 |
| 征地要改革 | 现行征地制度损害农民利益，需要根本性改革 |
| 土地要入市 | 集体建设用地应直接进入市场，打破政府垄断 |
| 开放促改革 | 对外开放倒逼国内改革，WTO是典型案例 |
| 地方有竞争 | 地方竞争是中国经济增长的重要动力 |

## 制度问题速查

| 问题 | 表现 | 改革方向 |
|------|------|----------|
| 户籍制度 | 城乡二元分割，农民工难以落户 | 放宽落户限制，实现同城同待遇 |
| 土地制度 | 城乡二元，集体土地不能入市 | 集体建设用地入市，同地同权 |
| 征地制度 | 公共利益泛化，补偿标准低 | 缩小征地范围，提高补偿标准 |
| 土地财政 | 依赖土地出让金，推高房价 | 房地产税改革，财权事权调整 |
| 社保制度 | 城乡分割，转移接续困难 | 全国统筹，转移接续 |

## 改革路径速查

### 户籍改革
```
路径：小城镇 → 中等城市 → 大城市 → 特大城市
方式：全面放开 → 有序放开 → 合理控制
配套：就业、教育、医疗、住房、社保
```

### 土地改革
```
路径：确权 → 流转 → 入市 → 同权
重点：集体建设用地入市、宅基地制度改革
目标：同地同权同价
```

### 城镇化模式
```
方向：以人为核心的新型城镇化
重点：城市群为主体形态
目标：3亿农民工市民化
```

## 常见问答速查

**Q: 中国城市化率是否过高？**
A: 不高，反而滞后。与同等发展水平国家相比，中国城市化率偏低约10个百分点。

**Q: 为什么农民工不愿意放弃农村户口？**
A: 因为土地是最后的保障，城市公共服务又不均等，放弃农村户口意味着失去土地权益。

**Q: 集体土地入市会不会冲击耕地保护？**
A: 不会。入市的是集体建设用地，不是耕地。而且可以通过规划管控确保耕地红线。

**Q: 地方竞争有什么负面影响？**
A: 重复建设、产能过剩、环境污染、地方保护、债务风险等。

**Q: 为什么改革总是渐进式的？**
A: 因为改革涉及利益调整，渐进式可以降低阻力、允许试错、积累经验。

## 重要概念对照

| 概念 | 含义 | 关联概念 |
|------|------|----------|
| 城市化 | 人口向城市集中 | 城镇化、市民化 |
| 市民化 | 农民工获得城市身份 | 户籍改革、公共服务 |
| 土地财政 | 地方政府靠卖地收入 | 土地出让金、房地产税 |
| 增减挂钩 | 农村建设用地减少换城市指标 | 土地整理、占补平衡 |
| 同地同权 | 集体土地与国有土地同等权利 | 集体土地入市 |
| 同城同待遇 | 户籍人口与常住人口同等服务 | 基本公共服务均等化 |
