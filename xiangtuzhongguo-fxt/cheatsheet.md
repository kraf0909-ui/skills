# 速查手册（Cheatsheet）

## 核心概念对照

| 乡土社会 | 现代社会 | 关键区别 |
|----------|----------|----------|
| 差序格局 | 团体格局 | 以己为中心 vs 团体界限分明 |
| 礼治秩序 | 法治秩序 | 传统教化 vs 国家权力 |
| 长老权力 | 时势权力 | 经验权威 vs 变革权威 |
| 血缘结合 | 地缘结合 | 身份社会 vs 契约社会 |
| 欲望行事 | 需要计划 | 文化习惯 vs 理性计算 |
| 亚普罗式 | 浮士德式 | 稳定维持 vs 创造变化 |
| 无讼理想 | 维权诉讼 | 教化羞耻 vs 权利正当 |
| 无为政治 | 有为政治 | 经济限制 vs 主动治理 |

## 四种权力速查

| 权力类型 | 来源 | 性质 | 基础 | 典型场景 |
|----------|------|------|------|----------|
| 横暴权力 | 社会冲突 | 压迫性 | 剥削 | 统治阶级 |
| 同意权力 | 社会合作 | 契约性 | 分工 | 民主政治 |
| 教化权力 | 社会继替 | 教化性 | 传统 | 父子师徒 |
| 时势权力 | 社会变迁 | 创造性 | 变革 | 革命/现代化 |

**判断规则**: 看权力关系发生在什么情境中 → 确定权力类型 → 理解行为逻辑。

## 差序格局应用规则

### 当分析中国社会现象时:
```
1. 找到"己"的中心位置
2. 画出关系圈层（亲疏远近）
3. 判断行为标准如何随圈层变化
4. 注意公私界限的伸缩性
5. 识别自我主义（非个人主义）的表现
```

### 常见误读警示:
- 把"私"简单理解为道德败坏 → 实际是结构特征
- 用团体格局标准评判差序格局 → 概念错位
- 认为中国没有道德 → 实际是没有"普遍"道德

## 礼治vs法治决策树

```
社会变迁速率如何?
  ├─ 很慢（三代不变）→ 礼治可能
  │   └─ 传统是否仍有效?
  │       ├─ 是 → 礼治秩序
  │       └─ 否 → 注释/名实分离
  └─ 很快 → 法治需要
      └─ 是否有共同同意的规则?
          ├─ 是 → 法治秩序
          └─ 否 → 混乱/时势权力崛起
```

## 乡土社会诊断清单

判断一个社区是否仍具乡土性:
- [ ] 人口是否高度稳定（世代定居）
- [ ] 是否是熟人社会（无陌生人）
- [ ] 纠纷是否通过调解而非诉讼解决
- [ ] 权力是否由长者/权威主导
- [ ] 经济是否以农业/小农为主
- [ ] 社会关系是否以血缘为核心
- [ ] 道德标准是否因对象而异
- [ ] 对文字/法律的依赖程度是否低

**评分**: 勾选越多，乡土性越强。

## 社会转型分析框架

### 当分析中国从传统到现代转型时:

**结构层面**:
- 差序格局是否在松动？
- 团体意识是否在形成？
- 公私界限是否在明确化？

**秩序层面**:
- 礼治秩序是否仍在维持？
- 法治秩序是否有效建立？
- 两者是否冲突？

**权力层面**:
- 长老权力是否仍有效？
- 时势权力是否崛起？
- 青年是否挑战长者权威？

**结合层面**:
- 血缘关系是否仍主导？
- 商业/契约关系是否发展？
- 陌生人信任是否建立？

## 费孝通研究方法论速查

| 方法 | 应用场景 | 操作要点 |
|------|----------|----------|
| 社区研究 | 理解具体社会 | 深入一个社区，全面观察 |
| 类型比较 | 提炼理论概念 | 建立理想类型，对照现实 |
| 功能分析 | 理解制度存在理由 | 问"这制度对社会有什么作用" |
| 中西比较 | 认识中国特色 | 避免用西方概念套中国现实 |
| 历史比较 | 理解社会变迁 | 从传统到现代的演变分析 |

## 常见问答速查

**Q: 为什么乡下人不识字不是"愚"？**
A: 因为乡土社会是面对面社群，文字是多余的。识字能力取决于环境需求，不是智力问题。

**Q: 差序格局中的"私"是道德问题吗？**
A: 不是个人道德败坏，而是社会结构特征。在差序格局中，公和私是相对的，站在任何一圈向内看都是公的。

**Q: 为什么乡土社会追求"无讼"？**
A: 因为礼治秩序中，诉讼意味着教化失败。理想状态是每个人自动守规矩，不需要外在监督。

**Q: 长老权力为什么不容反对？**
A: 因为教化权力建立在"有知对无知"上。如果文化有效，被教者无需反对；如果失效，教化本身失去意义。

**Q: 为什么单把法律推行下乡会破坏秩序？**
A: 因为破坏了礼治秩序的基础，但人民不会应用法律设备，社会结构和观念未改革，法治秩序无法有效建立。
