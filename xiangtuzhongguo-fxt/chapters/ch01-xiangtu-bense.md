# 第一章：乡土本色

## Core Idea
中国社会的基层是乡土性的。理解中国社会，必须首先理解其乡土根基。乡下人离不了泥土，因为农业直接取资于土地，而土地是不流动的。这种人与土地的紧密绑定，塑造了中国人世代定居的生活常态，也构成了中国社会最基本的文化底色。

## Frameworks Introduced
- **礼俗社会 vs 法理社会（Gemeinschaft vs Gesellschaft）**：借用滕尼斯的概念，费孝通指出乡土社会是"礼俗社会"——一种由共同生长、亲密熟悉、默认遵守传统规范而结合的社会；而现代社会是"法理社会"——一种由目的、任务、理性契约而结合的社会。前者是"有机的团结"，后者是"机械的团结"。
- **熟悉的社会**：乡土社会是一个没有陌生人的社会。在这种社会里，信任不是建立在契约之上，而是建立在"熟悉"之上——从熟悉里产生的信任与规矩。

## Key Concepts
- **乡土性**：中国社会基层的本质特征，表现为对土地的依附、定居常态、熟人网络。
- **土气**：并非贬义，而是乡土社会与土地不可分离的生活气质。乡下人离不了泥土，正如城里人离不开空气。
- **世代定居**：乡土社会中，人口附着于土地，迁移是反常，定居是常态。"生于斯，死于斯"是普遍现象。
- **熟悉**：乡土社会的人际关系基础。在熟人中，规矩不是法律，而是"习"出来的礼俗。

## Mental Models
- **从土地出发理解社会结构**：一切社会特征都可以追溯到人与土地的关系。土地的不流动性决定了人的不流动性，人的不流动性决定了社会的封闭性与熟人属性。
- **"面对面"社群模型**：在小型、封闭、稳定的社群中，全部信息可以通过直接接触传递，无需借助抽象符号系统。
- **传统即正当**：在礼俗社会中，"历来如此"本身就是行为正当性的来源，不需要进一步论证。

## Anti-patterns
- **以城市/现代视角评判乡下人**：认为乡下人"土气"是落后、愚昧的表现，这是一种文化中心主义的偏见。乡土社会的特征是其生存环境塑造的合理适应。
- **将乡土性等同于落后性**：乡土社会的"不流动"并非缺陷，而是与其生产方式相匹配的社会形态。
- **忽视社会基层的独立性**：用上层政治结构或现代法律制度去直接解释基层社会行为，忽视乡土社会自有的运行逻辑。

## Key Takeaways
1. 中国社会的基层是乡土性的，这是理解一切中国问题的起点。
2. 乡土社会的核心特征是"不流动"——人附着于土地，形成世代定居的熟人社会。
3. 乡土社会是"礼俗社会"（Gemeinschaft），靠熟悉和传统维持秩序，而非靠契约和法律。
4. 在乡土社会中，"规矩"不是外在的强制法律，而是内化于日常习惯的礼俗。
5. 对乡土社会的任何评判，都应首先回到其自身的生存逻辑，而非套用现代法理社会的标准。
