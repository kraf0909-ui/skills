# 第六章：家族

## Core Idea
中国乡土社会的基本社群是"小家族"（而非"大家庭"）。西洋家庭是暂时性的生育社群，中国家族是长期性的事业组织。家族的主轴在父子、婆媳之间（纵向的），夫妇只是配轴。事业需要纪律，因此排斥私情。

## Frameworks Introduced
- **西洋家庭模型**：以夫妇关系为核心，是暂时性的生育社群。孩子成年后即独立成家，家庭生命周期有限。夫妇感情是家庭的主轴，家庭的主要功能是生育和情感满足。
- **中国家族模型**：以父子关系为核心，是长期性的事业组织。家族不限于生育功能，而是承担政治、经济、宗教、教育等多重功能。家族是一个绵延性的事业社群，不因个别成员的生死而终结。
- **主轴与配轴**：
  - **主轴（纵向）**：父子关系、婆媳关系。这是家族延续和事业传承的核心轴线，强调纪律、服从、责任。
  - **配轴（横向）**：夫妇关系。在事业组织中，夫妇关系是辅助性的，服务于家族延续的大业，而非家庭的核心。

## Key Concepts
- **小家族**：费孝通用以区分于"大家庭"的概念。不是指规模小，而是指其性质——它是一个以父子为主轴的事业组织，而非以夫妇为主轴的生育单位。
- **事业组织**：家族的功能远超生育。它是乡土社会中的经济生产单位、政治权力单位、宗教祭祀单位和教育教化单位。
- **绵延性**：家族不以个人生命周期为限，而是代代延续的。个人的死亡不影响家族的存续，因此个人必须服从于家族的长期利益。
- **纪律与私情**：事业组织需要效率，效率需要纪律。而夫妇之间的私情可能破坏纪律（如偏爱、徇私），因此家族结构有意抑制夫妇感情的发展。
- **父子主轴**：父子关系是家族结构的核心。父亲代表权威、传统和事业，儿子代表继承、延续和未来。

## Mental Models
- **从功能理解结构**：家族的结构（父子主轴、夫妇配轴）不是偶然的，而是由其"事业组织"的功能所决定的。结构服务于功能。
- **个人作为家族的环节**：在绵延性的家族中，个人不是独立完整的单位，而是家族链条上的一个环节。个人的价值在于其对家族延续的贡献。
- **情感与效率的权衡**：事业组织追求效率，而情感（尤其是两性情感）具有不确定性和排他性，可能干扰组织目标。因此家族结构选择抑制私情以保障效率。

## Anti-patterns
- **用西洋家庭概念理解中国家族**：将中国的"家"理解为以夫妇感情为核心的"家庭"，会完全误解其结构和功能。
- **将家族压抑情感视为"封建残余"**：夫妇关系的淡化不是道德堕落，而是事业组织的结构性要求。
- **忽视家族的"事业"属性**：只看到家族的亲属关系网络，而忽视其作为经济、政治、宗教综合体的功能。

## Key Takeaways
1. 中国乡土社会的基本社群是"小家族"——一个长期性、绵延性的事业组织，而非暂时性的生育社群。
2. 家族的主轴在父子、婆媳之间（纵向），夫妇只是配轴（横向）。
3. 家族承担政治、经济、宗教、教育等多重功能，是乡土社会的核心组织单位。
4. 事业组织需要纪律，因此家族结构有意抑制夫妇私情，以保障效率和延续。
5. 个人在家族中不是独立单位，而是家族链条上的一个环节，其价值在于对家族延续的贡献。
