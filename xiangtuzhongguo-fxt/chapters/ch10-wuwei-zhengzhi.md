# 第十章：无为政治

## Core Idea
权力有两种基本类型：横暴权力（基于冲突与压迫）和同意权力（基于合作与契约）。在农业社会中，横暴权力受经济条件严格限制——农业剩余少，过度榨取会激起民怨、破坏税基。因此皇权"有为"则民怨沸腾，"无为"则休养生息，形成历史循环。乡土社会中，小农经济自给自足，同意权力的范围也极小。整体而言，乡土社会的权力结构松弛微弱，呈现"无为"状态。

## Frameworks Introduced
- **横暴权力（Coercive Power）**：基于冲突关系产生的压迫性权力
  - When to use：分析国家、统治阶级对被统治阶级的强制力
  - How：识别权力来源——暴力垄断、资源控制、制度压迫
- **同意权力（Consensual Power）**：基于合作关系产生的契约性权力
  - When to use：分析社会分工中产生的授权与服从关系
  - How：识别权力来源——社会契约、分工需要、共同利益
- **无为政治**：乡土社会权力结构松弛微弱的状态
  - When to use：解释中国传统政治中"休养生息""轻徭薄赋"的周期性出现
  - How：农业剩余有限 → 横暴权力扩张受限 → 权力结构松弛 → 社会呈现"无为"状态

## Key Concepts
- **横暴权力**：从社会冲突中产生的、具有压迫性质的权力，体现为统治与被统治的关系
- **同意权力**：从社会合作中产生的、基于契约和授权的 power，体现为社会分工中的权利让渡
- **农业剩余**：农业生产中超出农民自身消费需要的剩余产品，是横暴权力榨取的物质基础
- **皇权有为**：皇帝积极作为、大兴土木、开疆拓土、加重赋役的统治状态
- **皇权无为**：皇帝轻徭薄赋、与民休息、减少干预的统治状态
- **历史循环**：皇权在"有为"（民怨）与"无为"（养民）之间周期性摆动的历史规律

## Mental Models
- **农业剩余 = 权力的天花板**：横暴权力的扩张受限于农业经济能提供的剩余总量，超过这个天花板，统治就会崩溃
- **权力的成本收益计算**：统治者使用权力是有成本的（维持军队、官僚系统），当榨取成本超过收益时，"无为"成为理性选择
- **小农经济的"原子化"与权力的"真空"**：小农自给自足，社会分工简单，同意权力缺乏生长土壤，导致基层权力结构稀薄

## Anti-patterns
- **将"无为"等同于"无能"或"懒惰"**：不理解"无为"是特定经济结构下权力运行的理性状态
- **用现代国家能力衡量传统政治**：以现代官僚国家的高渗透性为标准，批评传统国家"治理能力低下"
- **忽视同意权力的经济前提**：以为同意权力可以脱离社会分工和合作需求而独立存在

## Key Takeaways
1. 横暴权力和同意权力是理解权力现象的两种基本分析工具
2. 农业社会的低剩余决定了横暴权力的扩张有严格的经济上限
3. 中国历史上"盛世"往往与"无为"政策相关，"乱世"往往与"有为"过度相关
4. 乡土社会的小农经济自给自足，使得同意权力的范围也极为有限
5. "无为政治"不是理想设计，而是农业经济条件下的结构性结果

## Connects To
- **第十一章（长老统治）**：在国家权力"无为"的乡土社会中，长老权力填补了权力真空，成为实际的秩序维持者
- **第八章（礼治秩序）**：礼治秩序的维持不需要国家权力的积极介入，与"无为政治"形成互补
- **第十二章（血缘和地缘）**：血缘社会的自我封闭性进一步减少了国家权力介入的必要性
- **第十三章（名实的分离）**：当社会变迁加剧，原有的无为政治无法应对新问题时，时势权力开始兴起
