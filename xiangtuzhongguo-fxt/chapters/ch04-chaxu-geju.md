# 第四章：差序格局

## Core Idea
中国社会的"私"问题，根源在于社会结构的不同。西洋社会是"团体格局"（如捆柴），界限分明；中国社会是"差序格局"（如水波纹），以"己"为中心，一圈圈推出去。这种结构产生了"自我主义"（非个人主义）——克己复礼，从己到家、到国、到天下。

## Frameworks Introduced
- **团体格局（捆柴模型）**：西洋社会的基本结构。如同一捆一捆扎清楚的柴，团体有明确的边界，团体中的分子（个人）是平等的。个人是团体的基本单位，团体超越个人而存在。
- **差序格局（水波纹模型）**：中国社会的基本结构。如同石头丢进水面激起的波纹，以"己"为中心，一圈圈推出去，愈推愈远，也愈推愈薄。没有明确的团体边界，关系随距离中心的远近而伸缩。
- **自我主义 vs 个人主义**：
  - **个人主义**：团体格局中的观念，主张个人权利，团体不能随意侵犯个人边界。
  - **自我主义**：差序格局中的观念，以"己"为中心，主张"克己复礼"，通过修身来影响外围的圈层。不是主张个人权利，而是主张以己为中心向外推展责任。

## Key Concepts
- **差序格局**：费孝通提出的核心概念，描述中国社会关系网络的基本形态。以己为中心，按亲疏远近形成有差等的关系圈层。
- **团体格局**：西洋社会结构，团体边界清晰，团体内个人平等，团体对个人的权利义务有明确规定。
- **私**：中国社会中"私"的问题，不是因为中国人道德低下，而是因为差序格局中缺乏超越差序的普遍标准——公与私的界限随中心位置而伸缩。
- **克己复礼**：儒家修身路径，从约束自己开始，推己及人，最终影响家国天下。这是差序格局中的行动逻辑。
- **推**：差序格局的核心动词。关系不是固定结构，而是动态推展的过程——从己到家、到国、到天下。

## Mental Models
- **水波纹模型**：理解中国社会关系的最核心隐喻。每个人都是水波纹的中心，关系随距离而衰减，没有绝对的边界。
- **可伸缩的公私边界**：在差序格局中，"公"与"私"不是固定概念，而是相对于"己"的位置而言。当为自己谋利是"私"，为家族谋利可能就是"公"；为家族是"私"，为国家谋利又可能是"公"。
- **中心-边缘思维**：一切价值判断、道德义务、资源分配，都以与中心的距离为标尺。

## Anti-patterns
- **用团体格局的标准批判中国人的"私"**：认为中国人"自私"是道德缺陷，忽视了这是差序格局结构的必然产物。
- **将自我主义等同于个人主义**：二者结构完全不同。个人主义有明确的权利边界，自我主义没有固定边界，只有可伸缩的差序。
- **寻找"中国社会中的团体"时套用西洋概念**：中国的"家""国"并非西洋意义上的团体，而是差序格局中可伸缩的圈层。

## Key Takeaways
1. 西洋社会是"团体格局"（捆柴），界限分明；中国社会是"差序格局"（水波纹），以己为中心向外推展。
2. 差序格局中没有固定的团体边界，关系随与中心的距离而伸缩，愈推愈薄。
3. 中国社会中的"私"问题，根源在于差序格局缺乏超越差序的普遍标准，而非个人道德缺陷。
4. 差序格局产生的是"自我主义"——以己为中心，克己复礼，推己及人——而非西洋的"个人主义"。
5. "推"是差序格局的核心机制：从己到家、到国、到天下，每一圈的关系性质和义务标准都不同。

- **第六章《家族》**：差序格局中的"家"是一个可伸缩的事业组织，而非固定边界的生育社群。
