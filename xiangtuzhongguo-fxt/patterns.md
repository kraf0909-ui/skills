# 模式与框架（Patterns & Frameworks）

## 1. 社会结构二元分析框架

```
社会结构 = 团体格局 vs 差序格局

团体格局（西洋）:
- 像捆柴，界限分明
- 个人是团体分子，地位平等
- 道德观念超越个人（神/国家）
- 权利义务分明

差序格局（中国乡土）:
- 像水波纹，以己为中心
- 自我主义，无团体前提
- 道德在私人联系中
- 公私相对，界限伸缩
```

## 2. 社会秩序类型分析框架

```
社会秩序类型:

礼治秩序（乡土社会）:
- 规范: 礼（传统）
- 维持力量: 教化、习惯、内心敬畏
- 人的态度: 主动服膺
- 适用条件: 变迁缓慢、传统有效
- 理想状态: 无讼

法治秩序（现代社会）:
- 规范: 法律
- 维持力量: 国家权力
- 人的态度: 被动遵守（外在限制）
- 适用条件: 变迁快速、需要明确规则
- 特征: 权利本位

人治（误解）:
- 不是"任凭好恶"
- 礼治常被误称为人治
```

## 3. 权力类型四象限

```
权力来源 → 社会冲突 | 社会合作 | 社会继替 | 社会变迁
权力名称 → 横暴权力 | 同意权力 | 教化权力 | 时势权力
性质     → 压迫性   | 契约性   | 教化性   | 创造性
基础     → 剥削     | 分工     | 传统     | 变革
典型场景 → 统治阶级 | 民主政治 | 父子师徒 | 革命/现代化
```

**应用方法**: 分析任何社会现象时，先判断权力类型，再理解其行为逻辑。

## 4. 社会变迁分析框架

```
社会变迁速率 → 慢          | 快
权力主导     → 长老权力    | 时势权力
秩序维持     → 礼治        | 法治
行为指导     → 欲望        | 需要
社会结合     → 血缘        | 地缘
文化模式     → 亚普罗式    | 浮士德式
```

**关键判断**: "三年无改于父之道"——变迁速率慢到可与世代交替相等时，社会是安定的。

## 5. 乡土社会特征矩阵

| 维度 | 乡土社会 | 现代社会 |
|------|----------|----------|
| 空间关系 | 不流动，定居 | 流动，迁移 |
| 人际关系 | 熟人，面对面 | 陌生人，间接 |
| 社会结构 | 差序格局 | 团体格局 |
| 秩序类型 | 礼治 | 法治 |
| 权力主导 | 长老权力 | 时势权力/同意权力 |
| 道德基础 | 私人关系 | 普遍原则 |
| 行为指导 | 欲望 | 需要 |
| 社会结合 | 血缘 | 地缘/契约 |
| 感情定向 | 亚普罗式（稳定） | 浮士德式（创造） |
| 文字功能 | 多余 | 必需 |
| 诉讼观念 | 无讼（羞耻） | 维权（正当） |
| 政治理想 | 无为 | 有为/计划 |

## 6. 名实分离分析模式

```
当社会变迁加速时:
传统形式（名）→ 不准反对
实际内容（实）→ 通过注释改变
结果 → 名实分离
表现 → 口是心非、面子文化、虚伪必需
```

**应用**: 分析中国传统社会中的"阳奉阴违"、政策执行偏差等现象。

## 7. 从血缘到地缘的转型路径

```
血缘社会 → 人口繁殖 → 土地限制 → 社群分裂
                ↓
         向外开垦 → 新村落 → 保持血缘联系
                ↓
         无法开垦 → 插入其他社区 → 成为"客边"
                ↓
         商业发展 → 陌生人交易 → 契约关系 → 地缘社会
```

**关键洞察**: 地缘是从商业里发展出来的社会关系，地缘结合是契约社会的基础。

## 8. 乡土社会向现代社会转型障碍分析

```
转型障碍:
1. 结构层面: 差序格局 vs 团体格局 → 公私界限模糊
2. 秩序层面: 礼治 vs 法治 → 法律与伦理冲突
3. 权力层面: 长老权力 vs 时势权力 → 代际冲突
4. 认知层面: 欲望 vs 需要 → 缺乏计划理性
5. 结合层面: 血缘 vs 地缘 → 商业发展受限

转型路径:
- 不是简单的"现代化"
- 需要社会结构和思想观念同步改革
- 单把法律推行下乡 → 破坏礼治而法治未立
```

## 9. 四种权力类型的应用场景

### 横暴权力
**When to use**: 分析阶级关系、统治结构、压迫机制
**How**: 看谁是权力的受益者，谁是被剥削者
**Example**: 皇权与农民、殖民者与被殖民者

### 同意权力
**When to use**: 分析民主政治、社会契约、分工体系
**How**: 看权力是否基于被统治者的同意
**Example**: 现代民主国家、行业协会

### 教化权力（长老权力）
**When to use**: 分析传统社会、教育关系、文化传承
**How**: 看权力是否基于"有知对无知"的教化
**Example**: 父子关系、师徒关系、传统村落

### 时势权力
**When to use**: 分析社会变革、革命运动、现代化过程
**How**: 看是否有"文化英雄"提出新方案
**Example**: 革命领袖、改革者、创新者

## 10. 差序格局中的道德应用

```
在差序格局中应用道德:
- 没有统一标准
- 标准取决于: 对象是谁 + 与自己的关系
- 同样行为，对不同对象有不同评价

Example:
- 父亲贪污 → 儿子代讳隐（孝）
- 他人贪污 → 痛骂（义）
- 自己贪污 → "能干"（智）

这不是矛盾，而是差序逻辑的自然结果。
```

## 11. 乡土社会研究方法论

```
费孝通方法论要点:
1. 社区研究法: 从具体社区入手，深入实地调查
2. 比较分析法: 中西比较（如差序格局vs团体格局）
3. 类型学方法: 提炼理想类型（礼俗社会vs法理社会）
4. 功能分析法: 看制度对社会完整的作用
5. 历史比较法: 从传统到现代的变迁分析

研究原则:
- 不套用西方概念，从中国实际出发
- 注重"型式"（pattern）而非个案
- 理解重于评价
```
