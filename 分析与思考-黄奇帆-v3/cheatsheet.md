# 快速参考与决策指南

## 一、核心决策规则

### 何时启动重组思维？
| 场景 | 判断标准 | 行动 |
|------|----------|------|
| 超常规发展 | 常规方法无法实现目标 | 启动重组，打破常规 |
| 历史遗留问题 | 长期存在、多轮未解决 | 创新重组，跳出传统坐标 |
| 广泛共性问题 | 多方尝试均未成功 | 独特重组思维 |
| 危机性问题 | 爆发性、冲撞性事件 | 紧急重组，抓住机遇 |
| 体制机制变革 | 深层次结构性矛盾 | 系统性重组 |

### 问题—结构—对策分析速查
1. 问题是什么？（现象描述）
2. 结构性短板在哪里？（供需结构、产业结构、区域结构）
3. 体制性障碍是什么？（所有制、产权、市场准入）
4. 机制性缺陷是什么？（价格、竞争、激励）
5. 制度性缺失是什么？（法律、规则、标准）
6. 改变什么边界条件可以化解？

## 二、政策分析速查表

### 房地产调控决策树
```
房价过高？
├── 是 → 土地供给是否充足？
│   ├── 否 → 增加土地供给（地票制度）
│   └── 是 → 金融杠杆是否过高？
│       ├── 是 → 控制房贷比例、提高首付
│       └── 否 → 是否存在投机？
│           ├── 是 → 税收政策调节（房产税预期）
│           └── 否 → 保障供给不足？
│               ├── 是 → 扩大保障房
│               └── 否 → 市场监管问题
└── 否 → 维持现状，防范风险
```

### 金融风险快速评估
| 指标 | 安全线 | 警戒线 | 危险线 |
|------|--------|--------|--------|
| 宏观杠杆率 | <250% | 250%-280% | >280% |
| 企业债务/GDP | <120% | 120%-150% | >150% |
| 房地产贷款占比 | <30% | 30%-40% | >40% |
| 不良贷款率 | <2% | 2%-5% | >5% |

### 资本市场改革优先级
1. **最优先**: 注册制 + 退市制度（根基）
2. **次优先**: 长期资本来源（年金、住房公积金改革）
3. **第三**: 投资者保护（分红、回购、反套现）
4. **第四**: 交易成本（印花税、交易费用）
5. **第五**: 市场结构（指数、证券公司功能）

## 三、关键数据速记

### 中国经济三大台阶
| 时间节点 | 目标 | 标志 |
|----------|------|------|
| 2025年 | 进入高收入国家行列 | 人均GNI跨越门槛 |
| 2030年左右 | 全球第一大经济体 | GDP超越美国 |
| 2035年 | 中等发达国家水平 | 基本实现现代化 |

### 重庆房地产调控经验
- 房价收入比控制在 **6~7**
- 地票制度满足建设用地需求
- 房地产投资占比控制在 **25%** 左右

### 中美脱钩风险分级
| 风险等级 | 领域 | 应对策略 |
|----------|------|----------|
| 高风险 | 科技、互联网、SWIFT | 自主创新、备用方案 |
| 中风险 | 贸易、投资、资本市场、教育 | 谈判博弈、多元化 |
| 低风险 | 金融机构、外汇、国际规则 | 保持沟通、预防为主 |

## 五、招商引资方式速选

| 方式 | 适用场景 | 关键成功因素 |
|------|----------|--------------|
| 产业链招商 | 已有核心企业 | 配套能力、产业生态 |
| 以商招商 | 有成功企业案例 | 口碑、示范效应 |
| 专业招商 | 特定产业领域 | 专业团队、行业资源 |
| 园区招商 | 有成熟园区 | 基础设施、服务体系 |
| 资本招商 | 有产业基金 | 资金实力、投资能力 |
| 人才招商 | 高科技产业 | 人才政策、创新环境 |
| 环境招商 | 综合竞争力强 | 营商环境、城市品质 |

## 七、要素市场化改革速查

| 要素 | 核心问题 | 改革方向 |
|------|----------|----------|
| 土地 | 城乡分割、配置低效 | 统一市场、市场化配置 |
| 劳动力 | 户籍限制、流动不畅 | 户籍改革、人才自由流动 |
| 资本 | 利率未完全市场化 | 利率市场化、资本市场改革 |
| 技术 | 转化难、产权不清 | 成果转化、知识产权保护 |
| 数据 | 产权不明、交易规则缺 | 产权界定、培育市场 |

## 八、常用判断准则

### "五个是否"判断改革成效
1. 是否提高了效率？
2. 是否促进了公平？
3. 是否激发了活力？
4. 是否防范了风险？
5. 是否增强了可持续性？

### "三个不能"底线思维
1. 不能发生系统性金融风险
2. 不能出现大规模失业
3. 不能引发社会动荡

### "四个坚持"原则
1. 坚持党的领导
2. 坚持社会主义市场经济方向
3. 坚持改革开放
4. 坚持以人民为中心
