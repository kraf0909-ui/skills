# Cheatsheet — Antifragile

## Decision Rules

### When to Intervene vs. Do Nothing
| Situation | Action | Why |
|-----------|--------|-----|
| Complex organic system | Wait, observe, minimal intervention | Iatrogenics risk is high |
| Mechanical system with known failure mode | Fix it | Low second-order effects |
| Someone else's problem | Ask if they have skin in the game | Avoid naive intervention |
| Crisis with information asymmetry | Delay (Fabian procrastination) | More info reduces error |
| Convex exposure to stress | Let it run | Volatility helps you |
| Concave exposure to stress | Protect, hedge, exit | Volatility hurts you |

### The Triad: Classify Everything
```
Is it harmed by stress?  → FRAGILE → Avoid, protect, or short
Is it neutral to stress?  → ROBUST  → Acceptable but not ideal
Is it helped by stress?   → ANTIFRAGILE → Seek, cultivate, invest
```

### Barbell Allocation
```
90% ultra-safe + 10% high-risk = Barbell
50% safe + 50% moderate = Mediocre (avoid)
```

## Decision Trees

### Should I Trust This Advice?
```
Does the advisor have skin in the game?
  ├─ Yes → Consider the advice
  │   └─ Do they bear cost if wrong?
  │       ├─ Yes → Higher trust
  │       └─ No → Lower trust
  └─ No → Dismiss or heavily discount
      └─ Are they a Fragilista?
          ├─ Yes → Ignore
          └─ Maybe → Verify with practitioners
```

### Is This System Truly Stable?
```
Has it been stress-tested recently?
  ├─ Yes → Moderate confidence
  │   └─ Did it improve from stress?
  │       ├─ Yes → Antifragile (best)
  │       └─ No → Robust (acceptable)
  └─ No → High suspicion (Turkey Problem)
      └─ Has stability lasted long?
          ├─ Very long → DANGER: confidence accumulates, risk hidden
          └─ Short → Too early to tell
```

## Trade-off Matrices

### Investment/Exposure Types
| Type | Upside | Downside | Volatility Effect | Preference |
|------|--------|----------|-------------------|------------|
| Convex (long options) | Unlimited | Fixed | Benefits | HIGH |
| Linear (cash) | None | Inflation | Neutral | MEDIUM |
| Concave (short options) | Fixed | Unlimited | Harms | AVOID |

### Knowledge Types
| Type | Source | Reliability | Action |
|------|--------|-------------|--------|
| Subtractive (via negativa) | Experience, time | High | Trust |
| Additive (theories) | Academia, models | Low | Verify |

## Thresholds & Defaults

### Lindy Effect Heuristic
- **< 5 years old**: Treat as experimental; high failure risk
- **5-50 years old**: Moderate confidence; observe survival
- **> 50 years old**: High confidence; likely to outlast new alternatives
- **> 500 years old**: Very high confidence (Stoicism, certain foods, etc.)

### Intervention Threshold
- **First, do no harm** (default)
- Intervene only when:
  1. Problem is mechanical, not organic
  2. You understand second-order effects
  3. You have skin in the game
  4. Inaction is clearly worse than action

### Portfolio Barbell Default
- 80-90%: Maximum safety (cash, Treasuries, stable skills)
- 10-20%: Maximum aggression (options, startups, moonshots)
- 0%: The middle (moderate-risk mutual funds, "balanced" portfolios)

## Tells & Smells

### Fragility Indicators
- Increasing confidence in predictions
- Long period without stress or failure
- Complex interdependencies
- Removal of slack and redundancy
- Centralization
- High debt/leverage
- "This time is different"

### Antifragility Indicators
- Benefits from small shocks
- Decentralized structure
- Redundancy and slack
- Optionality and flexibility
- Skin in the game throughout
- Long history of survival (Lindy)
- Respect for randomness

### Fragilista Detection
- Gives advice without bearing cost of error
- Uses complex models for complex reality
- Advocates for intervention as default
- Dismisses tail risks
- Works in bureaucracy, academia, or banking without personal stake
- Uses words like "optimal," "efficient," "rational" uncritically

## Quick Reference: Key Formulas

```
Antifragile = Convex exposure to volatility
Fragile     = Concave exposure to volatility
Robust      = Linear/neutral exposure

Optionality = Right, not obligation
            = Limited downside + Unlimited upside

Iatrogenics = Intervention harm > Intervention benefit
            → When in doubt, don't intervene

Lindy Effect: Age of non-perishable thing ∝ Expected remaining life

Via Negativa: Improvement = Removal of bad > Addition of good
```
