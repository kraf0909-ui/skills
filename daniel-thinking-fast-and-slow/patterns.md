# 模式与框架（Patterns & Frameworks）

## 1. 双系统思维框架

```
任何思维活动都可以归类为：

系统1（快思考）:
- 自动、快速、无意识
- 不费力、不消耗意志力
- 基于联想、直觉、情感
- 容易产生偏见和错误

系统2（慢思考）:
- 受控、缓慢、有意识
- 需要努力、消耗意志力
- 基于逻辑、分析、计算
- 可以纠正系统1的错误，但懒惰且容易疲劳

互动模式:
系统1不断提供建议 → 系统2通常接受 → 困难时系统2介入
```

**应用**: 分析任何判断或决策时，先判断是系统1还是系统2主导，再评估其可靠性。

## 2. 启发法-偏见分析框架

```
当面对复杂判断时，系统1会自动使用启发法：

代表性启发法:
- 问题: "A属于B的概率是多少？"
- 替换为: "A有多像B的典型代表？"
- 偏见: 忽视基础比率、合取谬误

可得性启发法:
- 问题: "B发生的频率是多少？"
- 替换为: "我能多容易想起B的例子？"
- 偏见: 媒体影响、个人经历偏差

锚定启发法:
- 问题: "这个数值是多少？"
- 替换为: "从这个锚点调整多少？"
- 偏见: 调整不足、受无关数字影响

情感启发法:
- 问题: "我对这件事怎么看？"
- 替换为: "我对这件事的感觉如何？"
- 偏见: 好感度影响风险评估
```

**应用**: 识别自己或他人的判断中使用了哪种启发法，然后检查可能的偏见。

## 3. 前景理论决策框架

```
人们如何做风险决策（偏离期望效用理论）：

参考点依赖:
- 结果被编码为收益或损失（相对于参考点）
- 不是最终财富状态

损失厌恶:
- 损失的痛苦 > 收益的快乐（约2-2.5倍）
- 导致风险规避（收益域）和风险寻求（损失域）

敏感性递减:
- 对小额变化的敏感 > 对大额变化的敏感
- 从$100到$200的差异感 > 从$1100到$1200

概率权重:
- 小概率被高估（买彩票、买保险）
- 中大概率被低估
- 确定性效应：从99%到100%的跳跃被特别重视

四重模式:
```

| | 高收益 | 低收益 |
|---|---|---|
| 高概率 | 风险规避（确定收益） | 风险规避 |
| 低概率 | 风险寻求（买彩票） | 风险寻求（买保险） |

## 4. 预测与判断改进框架

```
改进直觉预测的步骤:

1. 识别基础比率
   - 找出类似案例的统计结果
   - 不要从当前案例的特殊性出发

2. 评估证据质量
   - 区分有效预测因子和无效噪音
   - 警惕WYSIATI（你所见即全部）

3. 向平均值回归调整
   - 极端预测不可靠
   - 预测应比证据显示的更接近平均值

4. 使用外部意见
   - 参考类似项目的统计结果
   - 对抗内部意见的过度乐观

5. 考虑算法
   - 简单规则往往优于专家直觉
   - 除非环境有规律性且反馈即时
```

## 5. 幸福评估框架

```
两个自我的幸福评估:

经验自我（体验效用）:
- 问: "你现在感觉如何？"
- 测量: 体验的即时质量
- 原则: 持续时间重要

记忆自我（决策效用）:
- 问: "那段经历怎么样？"
- 测量: 峰终定律（高峰+结束）
- 原则: 持续时间被忽视

政策含义:
- 经验自我幸福: 减少痛苦时间、改善日常体验
- 记忆自我幸福: 创造高峰体验、美好结局
```

## 6. 风险政策设计框架

```
从窄框架到宽框架:

窄框架（系统1默认）:
- 孤立看待每个决策
- 对每个损失都感到痛苦
- 结果: 过度风险规避

宽框架（系统2努力）:
- 将多个决策视为投资组合
- 接受个别损失以换取整体收益
- 结果: 更理性的风险承担

应用:
- 个人: 将投资决策视为整体组合
- 组织: 事前验尸法、鼓励冒险文化
- 公共政策: 基于统计而非个案做决策
```

## 7. 专家直觉可信度评估

```
何时可以相信专家直觉:

可信条件（满足越多越可信）:
1. 环境有规律性（非随机）
2. 反馈即时且准确
3. 专家有大量刻意练习（>10000小时）
4. 判断领域有明确规则

不可信的情况:
1. 政治预测
2. 长期经济预测
3. 股市预测
4. 复杂社会现象

替代方案:
- 使用简单算法
- 采用结构化决策流程
- 汇集多个独立判断
```

## 8. 认知错觉识别与防御

```
常见认知错觉:

缪勒-莱耶错觉（视觉）:
- 两条等长线看起来不等长
- 防御: 用尺子量

认知错觉（思维）:
- 后见之明: "我早就知道"
- 防御: 记录事前预测

结果偏见:
- 以结果好坏评价决策质量
- 防御: 评估决策过程而非结果

叙事谬误:
- 将随机事件编织成因果故事
- 防御: 关注统计数据而非故事

最佳防御策略:
- 识别高风险情境
- 在重要决策时激活系统2
- 建立决策检查清单
- 寻求外部意见
```

## 9. 心理账户与消费决策

```
心理账户的类型:

当前收入账户:
- 最容易消费的账户
- 边际消费倾向最高

当前资产账户:
- 较难动用
- 用于大额消费或投资

未来收入账户:
- 最难消费
- 储蓄的主要来源

交易效用:
- 感知收益 = 获得效用 + 交易效用
- 交易效用 = 参考价格 - 实际价格
- 导致非理性消费（因为"划算"而购买不需要的东西）
```

## 10. 框架效应应用模式

```
框架效应的类型:

收益框架 vs 损失框架:
- "90%存活率" vs "10%死亡率"
- 收益框架导致风险规避
- 损失框架导致风险寻求

默认选项:
- 人们倾向于接受默认设置
- 应用: 器官捐赠默认同意、退休储蓄自动加入

措辞效应:
- "瘦肉80%" vs "肥肉20%"
- 同一事实的不同描述导致不同反应

积极应用（助推）:
- 设计选择架构帮助人们做出更好决策
- 不限制自由，只是改变呈现方式
```
