# 第11章 锚定效应在生活中随处可见

## Core Idea

锚定效应是实验心理学中最可靠也最稳健的结果之一。人们在对某一未知量的特殊价值进行评估之前，总会事先对这个量进行一番考量，此时锚定效应就会发生——估测结果和人们思考的结果很相近，就好比沉入海底的锚一样。卡尼曼和阿莫斯通过"幸运轮盘"实验揭示了这一效应的荒谬性：人们的判断明显受到没有任何信息价值的数字的影响。锚定效应有两种机制：调整（系统2的运行模式）和暗示/启动（系统1的自主显示模式）。调整通常会过早结束，因为当人们不再确定是否应该继续移动时就会停止。

## Frameworks Introduced

- **锚定效应（Anchoring Effect）**：人们在对某一未知量的特殊价值进行评估之前，总会事先对这个量进行一番考量，估测结果和人们思考的结果很相近，就像沉入海底的锚一样。
- **调整—锚定启发式（Adjustment-Anchoring Heuristic）**：从锚定的数字开始，然后估测它是过高还是过低，接着让大脑从锚定数值上"转移"并逐渐调整估值。调整通常会过早结束。
- **锚定指数（Anchoring Index）**：两个不同答案的比率。例如，高锚定值844英尺和低锚定值282英尺，差距562英尺，锚定指数为55%（562/1020）。
- **暗示作为锚定**：锚定的一种形式，由启发效应产生，是系统1的一种自主显示模式。

## Key Concepts

- **幸运轮盘实验**：
  - 轮盘指针只能停在10或65的位置上
  - 看到10的人对"非洲国家占联合国成员国的百分比"的平均估值为25%
  - 看到65的人的平均估值为45%
  - 轮盘的转动根本不可能为任何事情提供有用信息，但受试者没有忽略它的影响
- **调整不足的证据**：
  - 从纸的底端由下而上画一条2.5英寸长的线，与从顶端由上而下画一条线相比，前者往往被估测得更短
  - 开车下了高速公路驶入城市街道时还会开太快——调整不足
  - 孩子把音乐开得很大声，即使想调小也难——无法从充分的高锚定值中调整过来
- **点头与摇头的影响**：
  - 按照要求一听到锚定值就摇头的人仿佛对此有些排斥，会离锚定值更远
  - 点头则会增强他们对锚定值的概念
- **房地产经纪人实验**：
  - 经纪人坚称标价没有影响到她们的决策
  - 但实际上锚定指数是41%
  - 专业人士和对房地产一无所知的商学院学生一样，都受到了锚定效应的影响
  - 唯一区别：学生承认受到影响，专业人士否认
- **捐款锚定实验**：
  - 没有锚定问题时，平均捐款64美元
  - 锚定金额5美元时，平均捐款20美元
  - 锚定金额400美元时，平均捐款143美元
  - 锚定指数高于30%
- **德国法官的骰子实验**：
  - 法官掷骰子（提前做过手脚，结果不是3就是9）
  - 掷了9的法官平均判刑8个月
  - 掷了3的法官平均判刑5个月
  - 锚定指数是50%
- **限量购买的营销策略**：坎贝尔汤罐头促销时，"每人限购12罐"的平均购买量是不限购时的2倍。

## Mental Models

- **调整作为有意识的搜索**：调整就是刻意去寻找离开锚定数字的理由。但当人们的大脑资源耗尽时调整较少，离锚定较近。
- **锚定作为暗示/启动**：锚定有一种形式是由启发效应产生的——系统1试图建立一个将锚定数字视为真实数值的世界。这是联想一致性的一种表现。
- **锚定值的联想一致性**：大小不同的数字能激发起记忆中不同的观念体系。高锚定值使人想到夏天、奢华品牌；低锚定值使人想到冬天、大众品牌。
- **随机锚定的威胁**：随机锚定显然没有提供什么信息，但其强大影响是锚定效应的极端例子。你总能意识到锚定，甚至会对它格外关注，但你不知道它是如何引导和限制你的思考的。

## Anti-patterns

- **否认锚定效应的影响**：房地产经纪人坚称标价没有影响到她们的决策，但实际上锚定指数是41%。专业人士比学生更不愿承认受到影响。
- **调整不足**：从锚定数值上"转移"时，人们在不确定区域的边缘就会停止。调整不足是软弱或懒惰的系统2的一种失误。
- **先发锚定的谈判优势**：在协商中率先出击的一方往往会占有优势，但人们常常忽视这一点。
- **上限锚定的反效果**：给人身伤害案件的判决设置上限（如100万美元）会消除严厉判决，但锚定也会阻止法官将许多较轻罪行的量刑判得更轻。

## Key Takeaways

1. 锚定效应是实验心理学中最可靠也最稳健的结果——人们的判断明显受到没有任何信息价值的数字的影响。
2. 锚定效应有两种机制：调整（系统2的运行模式）和暗示/启动（系统1的自主显示模式）。
3. 调整通常会过早结束，因为当人们不再确定是否应该继续移动时就会停止。
4. 房地产经纪人的锚定指数是41%，专业人士和学生一样都受到了锚定效应的影响。
5. 德国法官的骰子实验中，掷了9的法官平均判刑8个月，掷了3的法官平均判刑5个月——锚定指数50%。
6. 限量购买是一种有效的营销策略——"每人限购12罐"的平均购买量是不限购时的2倍。
7. 在商谈中，有意地"为对方着想"的策略也许是抵制锚定效应的好方法。
