# 第18章 如何让直觉性预测更恰当有效？

## Core Idea

直觉性预测存在系统性偏见：预测与证据一样极端，缺乏回归性。人们用对证据的印象强度直接匹配预测值，忽略了预测中存在的不确定性。通过四个简单步骤——锚定平均值、评估证据匹配度、估计关联度、向平均值调整——可以修正直觉预测，使其更加恰当有效。

## Frameworks Introduced

- **直觉性预测的修正四步法**：
  1. 估测出结果的平均值（基准线）
  2. 根据对证据的印象算出与之相匹配的预测值（直觉预测）
  3. 对证据和结果的关联作出估计
  4. 按关联度从平均值向直觉预测调整
- **强度匹配（Intensity Matching）**：将证据的强度与预测量表上的值进行匹配，导致预测过于极端
- **基准线预测（Baseline Prediction）**：在仅知道案例所属类别、不了解具体信息时的预测
- **预认知决策模式（Recognition-Primed Decision）**：加里·克莱因描述的专家直觉决策过程

## Key Concepts

- **朱莉的阅读能力问题**：4岁能流畅阅读的朱莉，她的GPA会是多少？直觉预测（3.7-3.8）与证据一样极端，忽略了阅读年龄与大学成绩关联度很低（约0.3）
- **军官学校预测实验**：预测者将评级简单转换为军官学校评分，完全无法回归平均值——预测出的A和B频率几乎与实际结业成绩相同
- **预测中的替代**：人们将预测问题替换为对当前信息的评估问题，且未意识到这种替换
- **无偏见预测的特征**：只有在信息非常有效时才允许对罕见或极端事件作出预测，因此预测结果永远不会太离谱
- **修正的代价**：摆脱偏见意味着失去"我早就知道会这样"的愉快体验，也失去了对极端事件的预测能力

## Mental Models

- **共同因素与特殊因素**：阅读年龄 = 共同因素 + 决定阅读年龄的特殊因素；GPA = 共同因素 + 决定GPA的特殊因素
- **关联度决定调整幅度**：如果关联度是0.3，则从平均值中抽出30%放到直觉预测里
- **保守预测的价值**：对选手第二天的最准确预测通常是最保守、最接近平均值的

## Anti-patterns

- **预测与证据一样极端**：直觉性预测不具有回归性，导致对第一天发挥好的人过于乐观，对发挥差的人过于悲观
- **混淆评估与预测**：将对描述的印象强度直接当作对未来表现的预测
- **忽视不确定性**：系统1无法妥善处理预测中的不确定性，导致过度自信
- **拒绝适度预测**：风险投资家等需要极端语言来获得舒适感，即使这违背统计理性

## Key Takeaways

1. 直觉性预测需要校正是由于它并不具有回归性，因此带有偏见——预测与证据一样极端
2. 修正方法的核心：以平均值（基准线）为锚，根据证据与结果的关联度向直觉预测方向调整
3. 修正后的预测消除了过高或过低的系统性偏见，错误较小且不会导致极端估值
4. 修正直觉性预测是系统2的任务，需要付出很大努力，只有在风险很高时才显得合乎情理
5. 无偏见预测的一个特征：只有在信息非常有效时才允许对罕见或极端事件作出预测
