# 第5章 你的直觉有可能只是错觉

## Core Idea

系统1会自动评估认知放松度——介于"放松"和"紧张"之间的一个刻度。放松是事情进展顺利的标志，紧张说明存在问题。认知放松与多种输入和输出活动相联结：印刷清晰的文字、多次重复、好心情、咬笔"微笑"都会增加认知放松；而模糊字体、晦涩语言、皱眉则会增加认知紧张。认知放松导致直觉信任——我们更容易相信熟悉的事物，而认知紧张则激发系统2的怀疑。然而，系统2往往懒惰，倾向于采纳系统1的建议，这使得"真相错觉"成为普遍现象。

## Frameworks Introduced

- **认知放松度（Cognitive Ease）**：系统1自动评估的一个变量，介于"放松"和"紧张"之间。放松时人们心情好、相信直觉、感觉舒适熟悉；紧张时人们更警惕、多疑、投入更多精力。
- **真相错觉（Illusion of Truth）**：由认知放松导致的对熟悉事物的信任。想让人们相信谬误有个可靠的方法，那就是不断重复，因为人们很难对熟悉感和真相加以区别。
- **曝光效应（Mere Exposure Effect）**：罗伯特·扎伊翁茨提出，只要不断重复接触就能增加喜欢程度。重复曝光构成了社会组织和社会整合的基础。
- **认知反应测试（Cognitive Reflection Test, CRT）**：肖恩·弗雷德里克设计，包括球拍和球的问题、机器生产零件问题、睡莲叶子问题。
- **远隔联想测验（Remote Association Test, RAT）**：萨尔诺夫·梅德尼克设计，用于研究创造力——创新与出众的记忆力（极佳的联想记忆）有关。

## Key Concepts

- **认知放松的多重来源**：
  - 增加放松：印刷清晰、多次重复、已深植脑海、心情好、咬笔"微笑"
  - 增加紧张：印刷模糊、颜色浅、句子冗长、语言晦涩、心情不好、皱眉
- **一夜成名效应**：拉里·雅各比论证的记忆错觉——再次见到曾经见过的词时会更容易识别，正是这种放松使你产生了熟悉的感觉。
- **押韵增强可信度**：押韵的格言（"水滴石穿，绳锯木断"）比没有韵脚的版本（"再轻的击打也能放倒一棵大树"）显得更加深刻。
- **名字上口效应**：名字上口的经纪公司报告比名字拗口的更受信任；股票代码上口，回报率也相对较高。
- **字体模糊反而提高表现**：当问卷上的字体清晰时，90%的学生至少会犯一个错误；但当字体勉强能够辨认时，错误率降到了35%——认知紧张激发了系统2。
- **曝光效应的生理基础**：扎伊翁茨称，曝光效应的产生是因为一个刺激的重复曝光并没有产生不好的影响，这样的刺激最终会成为一个安全信号，而安全的就是好的。

## Mental Models

- **驾驶座舱刻度盘模型**：将大脑想象成飞机驾驶座舱，里面有一套刻度盘显示重要变量的当前数值。系统1自动对这些数值进行评估，决定是否需要系统2提供额外帮助。
- **认知放松-信任循环**：认知放松 → 心情好 → 喜欢亲眼所见的事物 → 相信亲耳所闻的消息 → 相信直觉 → 感到舒适熟悉 → 想法更随意肤浅。
- **认知紧张-怀疑循环**：认知紧张 → 更警惕多疑 → 对手头上的事情投入更多精力 → 感觉局促 → 较少犯错 → 直觉和创造下降。
- **系统1的"不可复返性"**：熟悉感有着简单而又强烈的"不可复返性"，似乎说明这种感觉是对过往经历的一种直接反应——但这是一种错觉。

## Anti-patterns

- **将熟悉感等同于真相**：人们很难对熟悉感和真相加以区别，这是权威机构和营销商都深谙的事实。
- **认知放松导致的轻信**：任何能使联想机制运行更轻松、更顺利的事物都会使我们心生偏见。
- **系统2的懒惰验证**：在大多数情况下，系统2都会采纳系统1的建议，并继续运作，而不去质疑信息的来源。
- **字体清晰度的反直觉效应**：人们以为信息呈现得越清晰越好，但实验表明字体模糊不清时学生的表现反而更好，因为认知紧张激发了系统2。

## Key Takeaways

1. 系统1会自动评估认知放松度，这个刻度盘介于"放松"和"紧张"之间。
2. 认知放松时，你有可能心情不错，会喜欢你亲眼所见的事物，相信你亲耳所闻的消息，相信你的直觉。
3. 认知紧张时，你更有可能警惕、多疑，会对手头上的事情投入更多精力，感觉局促，较少犯错。
4. 想让人们相信谬误有个可靠的方法，那就是不断重复——人们很难对熟悉感和真相加以区别。
5. 字体模糊不清时，学生的表现反而更好，因为无论认知紧张是由什么原因造成的，它都更有可能激发系统2来抑制系统1所给出的直觉性答案。
6. 曝光效应说明，只要不断重复接触就能增加喜欢程度，这是一个极其重要的生理现象。
7. 创新是发生在能让人联想无限的环境中的——创新与出众的记忆力（极佳的联想记忆）有关。
