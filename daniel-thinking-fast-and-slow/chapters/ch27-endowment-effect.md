# 第27章 禀赋效应与市场交易

## Core Idea

人们的偏好并非稳定不变，而是强烈依赖于参照点。一旦拥有某物，人们对其价值的评价会大幅提升，导致最低卖价远高于最高买价。传统经济学中的"无差异曲线"忽略了参照点的存在，而前景理论中的损失厌恶恰好解释了禀赋效应——放弃已拥有物品的痛苦远大于获得同样物品的快乐。

## Frameworks Introduced

- **禀赋效应（Endowment Effect）**：当个人一旦拥有某个物品，他对该物品价值的评价要比未拥有之前大大提高。
- **损失厌恶（Loss Aversion）**：一定量的损失比等量的所得影响更大，损失厌恶系数约为2:1。
- **参照点依赖（Reference Dependence）**：评估结果依赖于当前状态（参照点），而非绝对财富水平。
- **交易效用（Transaction Utility）**：交易中获得的效用不仅来自商品本身，还来自交易价格与参考价格的比较。

## Key Concepts

- **无差异曲线的缺陷**：传统经济学用无差异曲线表示偏好，但忽略了个人当前状态（参照点）的重要性。
- **快乐双胞胎实验**：艾伯特和本面对同样的选择，但因参照点不同而都不愿意交换，说明偏好随参照点变化。
- **R教授的葡萄酒**：买价不超过35美元，卖价却高达100美元，拥有状态改变了价值评估。
- **咖啡杯实验**：随机获得杯子的卖方定价（7.12美元）是买方定价（2.87美元）的两倍多，选择者定价（3.12美元）接近买方。
- **用来使用的商品 vs 用来交换的商品**：日常交易货币没有禀赋效应，但用于消费或享受的 goods（如葡萄酒、门票）会表现出强烈禀赋效应。
- **穷人没有禀赋效应**：贫困意味着生活水平低于参照点，穷人总是在损失中做选择。

## Mental Models

- **参照点即现状**：评估任何变化时，默认以当前状态为基准，不利之处比有利之处更突出。
- **商人思维**：经验丰富的商人学会问"与可得到的其他东西相比，这是我特别想得到的吗"，从而消除禀赋效应。
- **市场魔力有限**：标准市场理论预测半数代币会被转手，但杯子实验中实际交易量不到预测的一半。

## Anti-patterns

- **假设偏好稳定不变**：传统经济学假设人们的选择偏好不会随时间推移改变，忽略了参照点的作用。
- **忽视拥有状态对价值的影响**：标准理论认为拟定各项事务状态的效用只依赖于该状态本身，不受过往影响。
- **用财富状态而非变化来评估结果**：伯努利式的错误，忽略了得失相对于参照点的心理价值差异。

## Key Takeaways

1. 禀赋效应是行为经济学的里程碑发现，由理查德·泰勒借助前景理论解释。
2. 损失厌恶系数约为2:1，这一比例在多个经济领域研究中出现。
3. 禀赋效应可通过改变参照点而消除，经验丰富的交易者不会表现出禀赋效应。
4. 日常交易（货币交换）不产生禀赋效应，因为双方都将商品视为交换媒介而非使用品。
5. 价格变动的不对称性：价格上涨的效应（损失）约为收益效应的两倍。
