# 附录A 不确定性下的判断：启发法和偏见

## Core Idea

本文是卡尼曼和特沃斯基于1974年发表在《科学》杂志上的经典论文，系统阐述了人们在不确定性下进行判断时依赖的三种启发法（代表性、可得性、锚定）及其导致的系统性偏见。这些启发法在大多数情况下相当有用，但也会导致严重的、可预测的错误。文章揭示了即使是经验丰富的研究人员在凭直觉思考时，也易于犯这些偏见的错误。

## Frameworks Introduced

- **代表性启发法（Representativeness Heuristic）**：通过比较A与B的相似程度来评估概率。
- **可得性启发法（Availability Heuristic）**：通过能想到例子或事件的容易程度来评估频率或概率。
- **锚定与调整（Anchoring and Adjustment）**：通过初始值确定最后答案，但调整通常不充分。
- **主观概率（Subjective Probability）**：个人对不确定事件的信念程度，可通过赌注偏好推断。

## Key Concepts

### 代表性启发法导致的偏见

- **对先验概率（基础比率）不敏感**：史蒂夫像图书管理员的描述使人们忽视农民与图书管理员20:1的比例。
- **对样本大小不敏感**：认为大医院和小医院记录男婴比率>60%的天数大致相同，忽略了大样本更稳定。
- **误解机会（赌徒谬误）**：期望随机序列能代表过程的基本特征，如认为轮盘长时间红后该出黑了。
- **小数法则**：研究人员过于信任小样本结果，高估其可复制性。
- **对可预测性的不敏感**：预测仅依赖描述的好坏，不考虑证据的可靠性。
- **效度错觉**：因预测特点与输入信息吻合而产生的没有保证的自信。
- **误解回归性**：未能理解回归平均值现象，捏造虚假因果解释（如飞行训练中的表扬与批评）。

### 可得性启发法导致的偏见

- **因例子的可提取性导致的偏见**：名人更多的类别被认为数目更大；显著事件（如亲眼看到火灾）比读到的事件影响更大。
- **因搜索集合的有效性导致的偏见**：以r开头的单词被认为比第三个字母是r的单词更多，尽管后者实际更多。
- **想象力的偏见**：构建实例的容易程度不能反映真实频率（如委员会规模评估）。
- **相关性错觉**：高估自然联想物同时发生的频率（如疑心病与奇怪的眼睛）。

### 锚定与调整导致的偏见

- **不充分的判断**：幸运转盘的随机数字对后续估计产生巨大影响（非洲国家占联合国席位百分比：10→25，65→45）。
- **评估连续事件与非连续事件的偏差**：高估连续事件概率（计划成功），低估非连续事件概率（系统故障）。
- **评估主观概率分布时的锚定**：置信区间过小，反映过度确定性（30%的问题真实值落在X01-X99之外）。

## Mental Models

- **启发法即认知捷径**：人们在复杂判断中依赖简化的启发规则，这些规则通常有效但非最优。
- **系统1的自动运作**：启发法是系统1的快速、无意识运作特征，系统2很少介入纠正。
- **偏见是系统性的**：这些错误不是随机的，而是可预测、可重复的，源于启发法的结构性特征。
- **专家同样易犯错**：有经验的研究人员在凭直觉思考时，同样易于犯偏见的错误。

## Anti-patterns

- **忽视基础比率**：即使掌握了先验概率信息，人们在有具体描述时也会彻底忽略它。
- **信任小样本**：研究人员选择过小的样本量，并对仅有的发现作过多阐释。
- **捏造因果解释**：将回归平均值误解为因果效应（如飞行训练中的表扬与批评）。
- **过度自信**：设定的置信区间过小，反映了对自身知识的过度确定性。
- **从偏好推断概率**：现实中主观概率决定赌注偏好，而非从偏好中推断概率。

## Key Takeaways

1. 代表性启发法导致对基础比率不敏感、对样本大小不敏感、误解机会和回归性。
2. 可得性启发法导致因例子可提取性、搜索集合有效性和想象力产生的偏见。
3. 锚定效应导致调整不充分、连续/非连续事件评估偏差和概率分布校准不良。
4. 这些偏见不仅存在于普通人中，有经验的研究人员和专家在凭直觉思考时同样易犯。
5. 理性判断者应努力使概率判断与相关知识、概率规则以及个人的判断启发式和偏见相容。
