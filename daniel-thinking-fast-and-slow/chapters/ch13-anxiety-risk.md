# 第13章 焦虑情绪与风险政策的设计

## Core Idea

可得性效应能够解释买保险的行为模式和灾后的保护性行为模式：每次影响巨大的地震发生后，人们会去买保险、采取防护措施，但随着时间推移，对灾难的记忆变得模糊，担忧和防备的努力程度也会减弱。保罗·斯洛维克的研究表明，人们对风险的判断深受可得性和情感的影响——人们想到不同风险的轻松程度与其对这些风险的情感反应紧密相连。这种"概率忽视"和"效用层叠"的组合会导致公共政策中的优先处理权不稳定和错位：对小威胁的夸大有时还会引发严重后果。

## Frameworks Introduced

- **情绪启发式（Affect Heuristic）**：人们在作判断和决策时会受情绪的影响——"我喜欢它吗？我恨它吗？我对它的感觉有多强烈？"人们的观点和选择直接表达出其情感和取舍的基本倾向。
- **效用层叠（Availability Cascade）**：由卡斯·桑斯坦和蒂默尔·库兰提出，指一连串自持事件，可能开始于对相对次要的事件的媒体报道，然后引起公众恐慌和大规模的政府行动。
- **概率忽视（Probability Neglect）**：桑斯坦发明的短语，描述人们要么完全忽视风险，要么过于重视风险，没有中间地带的现象。
- **风险的主观性**：保罗·斯洛维克指出，"风险"并不是脱离我们的思想和文化而独立存在的，没有"真正风险"或"客观风险"这回事。风险评估依赖测试方法的选择。

## Key Concepts

- **灾后行为模式**：
  - 每次大地震后，加利福尼亚居民都会去买保险、采取防护措施
  - 随着时间推移，对灾难的记忆变得模糊，担忧和防备减弱
  - 记忆的动态变化解释了灾难、担忧和越来越松懈的心理循环
- **致死原因的可得性偏见**：
  - 中风致死的数量几乎是所有意外事故致死总数的2倍，但80%的受试者判断意外事故致死的可能性更大
  - 人们认为龙卷风比哮喘更容易致死，尽管后者的致死率是前者的20倍
  - 被闪电击中致死的概率是食物中毒的52倍，但人们认为前者更小
  - 意外死亡被认为是糖尿病致死率的300倍，但真正比率却是1:4
- **情绪启发式的运作**：
  - 人们想到不同风险的轻松程度与其对这些风险的情感反应紧密相连
  - 流畅生动的骇人印象会加深恐惧
  - 感性细节掌控理性大局
- **技术评价中的情感主导**：
  - 当人们更青睐某项技术时，就会认为此项技术更有优势、风险更小
  - 当不喜欢某项技术时，则只会想到其缺点和寥寥几个优点
  - 英国毒物学会会员的反应也相似：在自认为危险的物质中发现的优点很少
- **效用层叠的经典案例**：
  - 拉夫运河事件：1979年有毒垃圾显露，引发大规模恐慌和政府行动，但没有人能证实对健康造成了实质性损害
  - 艾拉恐慌（1989年）：苹果生长调节剂艾拉被报道可致癌，引发恐慌，苹果产业损失巨大，但后续研究证实致癌可能性很小
- **概率忽视与恐怖主义的对话**：恐怖袭击导致的死亡人数与其他死亡原因相比非常小，但媒体不断重复的可怕画面使每个人都处于崩溃边缘——恐怖主义是直接和系统1对话的。
- **专家vs公众的风险认知**：
  - 斯洛维克：公众对风险的认识比专家更深刻，专家不应忽略公众的恐惧情绪
  - 桑斯坦：专家是抵制"平民"越轨的壁垒，现行监管体系更多是对公众施压做出反应，而非谨慎客观分析

## Mental Models

- **灾难-担忧-松懈循环**：重大灾难 → 高度焦虑 → 买保险、采取防护措施 → 时间推移 → 记忆模糊 → 松懈 → 下一次灾难。
- **效用层叠的自我增强**：媒体报道 → 公众注意力 → 激愤和焦虑 → 更多媒体报道 → 更大焦虑 → 波及面更大 → 政府行动。
- **情感作为风险滤镜**：情绪启发式通过创造一个比现实更明了的世界来简化生活——好的技术在虚拟世界中成本较小，不好的技术没有利益。
- **概率忽视的"全有或全无"模式**：我们要么完全忽视风险，要么过于重视风险，没有中间地带。焦虑对伤害的可能性还不够敏感——你想到了分子，却没有想到分母。

## Anti-patterns

- **概率忽视**：等待晚归女儿的家长也许知道几乎没有什么可担心的，但大脑会不自觉地闪现危险的景象——想到了分子，却没有想到分母。
- **效用层叠的放大效应**：一个小问题通过媒体报道和公众恐慌被放大成重大危机，导致资源错配。
- **专家与公众的对立**：将专家从公众情感中隔离出来会产生公众排斥的政策；但完全顺从公众情绪又会导致监管体系的优先处理权错位。
- **可得性专业户的操纵**：专门负责连续不断地散布扰乱民心消息的个人或组织，刻意操纵效用层叠以推进其计划。
- **对小威胁的过度反应**：艾拉事件是对一个小问题做出的过激反应，对公众健康的最终影响可能是致命的——因为人们吃到的好苹果越来越少了。

## Key Takeaways

1. 每次大地震后人们都会去买保险、采取防护措施，但随着时间推移，对灾难的记忆变得模糊，担忧和防备减弱。
2. 中风致死的数量几乎是所有意外事故致死总数的2倍，但80%的受试者判断意外事故致死的可能性更大——可得性偏见在风险判断中作用巨大。
3. 情绪启发式认为人们在作判断和决策时会受情绪的影响——感性细节掌控理性大局。
4. 效用层叠是一连串自持事件，可能开始于对相对次要的事件的媒体报道，然后引起公众恐慌和大规模的政府行动。
5. 概率忽视描述了我们要么完全忽视风险，要么过于重视风险，没有中间地带的模式。
6. 恐怖主义是直接和系统1对话的——媒体不断重复的可怕画面可使每个人都处于崩溃的边缘。
7. 政策制定者必须努力保护公众不受恐惧情绪的影响，而不是只保护其不受真实存在的危险的伤害。
