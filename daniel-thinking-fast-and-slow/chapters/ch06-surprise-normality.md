# 第6章 意料之外与情理之中

## Core Idea

系统1的主要功能是维护并更新你个人世界的模式，呈现的都是常态下的思维模式。对惊喜的承受能力是精神生活的一个重要方面，而惊喜本身也是最敏感的指示，可以表明我们如何理解这个世界。当意外事件反复出现时，系统1会迅速将其纳入常态——第一次遇到心理学家乔恩是惊喜，两周后在伦敦剧院再次遇到他就变得"正常"了。系统1善于从惊讶到理解构建连贯的因果叙事，但这种能力也导致我们在随机事件中看到不存在的模式，并将因果性思考错误地用于需要统计论证的情景。

## Frameworks Introduced

- **常态理论（Norm Theory）**：由戴尔·米勒和卡尼曼提出，解释为什么我们觉得很多事情是正常或是不正常的。常态给我们提供了背景知识，使我们能够察觉到反常现象。
- **摩西错觉（Moses Illusion）**："摩西上方舟时，每种动物各带了多少？"——摩西并没有带动物上方舟，带动物上方舟的是诺亚。这个错觉说明系统1会迅速将不协调的信息纳入看似连贯的框架。
- **意向性归因（Intentional Attribution）**：人类生来就长于将行为和事件归因于意图和目的，即使面对无生命的物体（如海德的三角形和圆形实验）。
- **因果性直觉（Causal Intuition）**：人们总是很不恰当地将因果性思考用于需要统计论证的情景中。统计性思维总是根据事物的不同类别和总体性质得出个案的结论，但系统1并不具备这种推理能力。

## Key Concepts

- **从惊讶到习以为常**：第一次碰到乔恩很惊讶，第二次在伦敦剧院碰到就不那么惊讶了——系统1在不知不觉中改变了乔恩在我们大脑中的印象。
- **积极的期盼**：看到路旁有车着火，第二次遇到时就不那么惊讶了，甚至会产生积极的期盼——"每当走到这个路段时，我们都会想起着火的车"。
- **餐厅喝汤的例子**：第一位品汤的顾客往后退了一下，接下来服务生碰他时他吓了一跳、另一位顾客也喝了同样的汤后强忍着没有叫出来——这些事本应不正常，但因为重复了原来的情节，显得正常了。
- **摩西错觉的机制**：动物登上方舟给人们营造了《圣经》的文化背景，摩西也出现在这个背景中，对他的名字不会感到惊讶。摩西和诺亚两个名字的元音和音节数相同，容易使人混淆。
- **违反常态的惊人检测速度**：听到"地球每年绕着麻烦转"这个句子时，听到"麻烦"后不到0.2秒，大脑就会检测出特殊图谱。
- **简的钱包**："在纽约拥挤的大街上逛了一天，欣赏完美景后，简发觉自己的钱包丢了"——读者会认为"扒手"比"景色"与该句子联系更紧密，尽管后者在句子中出现了而前者没有。
- **米乔特的因果知觉实验**：1945年，阿尔伯特·米乔特通过黑色方块的动画证明，我们能像看见颜色那样直接"看到"因果现象。
- **海德的三角形电影**：大三角形、小三角形和圆形的动画让观看者看到气势汹汹的大三角形正欺负小三角形——意向性归因是系统1的产物。

## Mental Models

- **个人世界模式**：系统1维护并更新你个人世界的模式，这个模式由许多联想和结果共同构成，这些联想由情境、事件、行为等概念引发。
- **惊喜作为理解指示器**：惊喜本身是最敏感的指示，可以表明我们如何理解这个世界，我们希望从这个世界中得到什么。
- **因果叙事构建**：系统1熟练地将知识片段组合成一个连贯的因果关系。一件大事必然会带来一些后果，而这些后果也需要一些原因对其作出解释。
- **统计思维vs因果思维**：统计性思维要求同一时间把多件事情串联起来，而这一点系统1是做不到的。系统1善于因果思考，系统2通过学习可以进行统计性思考，但几乎没有人接受过必要的相关训练。

## Anti-patterns

- **在随机性中看到模式**：人们很容易在根本没有模式的情况下创建模式。二战期间伦敦火箭弹轰炸的分布被普遍认为不是随机的，但统计分析显示这正是随机程序的典型代表。
- **"投篮顺手"错觉**：运动员连续进球后被认为处于"投篮顺手"状态，但对上千个投篮动作的分析表明这完全是随机的。
- **过早的因果解释**：萨达姆被捕当天债券价格先涨后跌，媒体给出了完全相反的两个解释——系统1熟练地将知识片段组合成连贯的因果关系，哪怕这些解释互相矛盾。
- **将统计规律误认为因果规律**：某县人口稀少既不会引发癌症，也不能避免癌症，只会使癌症的发病率比人口稠密地方更高或更低——但人们倾向于寻找因果解释。

## Key Takeaways

1. 系统1的主要功能是维护并更新你个人世界的模式，呈现的都是常态下的思维模式。
2. 一个插曲如果反复出现就可能不那么令人惊喜了——系统1会迅速将新体验纳入常态。
3. 摩西错觉说明，在不知不觉中你就会将不协调的信息纳入看似连贯的框架。
4. 系统对违反常态的问题的察觉速度是惊人的——不到0.2秒就能检测出语义异常。
5. 我们生来就长于意向性归因——一岁以下的婴儿能辨别欺凌弱小的人和受害者。
6. 人们总是很不恰当地将因果性思考用于需要统计论证的情景中——系统1并不具备统计推理能力。
7. 在随机性中发现规律的想法往往不可逆转，肯定比某个人作了一项研究更有说服力。
