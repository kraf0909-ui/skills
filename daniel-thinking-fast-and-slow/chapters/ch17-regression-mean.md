# 第17章 所有表现都会回归平均值

## Core Idea

回归平均值（regression to the mean）是一种无处不在的统计现象：极端表现之后往往会跟随较不极端的表现，这与表扬、惩罚或任何干预措施无关。飞行教练将学员进步归因于怒吼、退步归因于表扬，实则只是随机波动的自然结果。人类大脑对回归现象陌生且抗拒，总是本能地寻求因果解释。

## Frameworks Introduced

- **回归平均值（Regression to the Mean）**：当两个测量值之间的关联不完美时，极端值后倾向于出现更接近平均值的值
- **相关系数（Correlation Coefficient）**：两个测量值共有因素的相对比重，取值在0到1之间
- **成功公式**：成功 = 天赋 + 运气；巨大成功 = 更多天赋 + 更多运气
- **体育画报诅咒**：登上封面的运动员接下来表现欠佳，实则是极端表现后的自然回归

## Key Concepts

- **飞行教练的谬误**：表扬后表现变差、惩罚后表现变好，不是因为表扬/惩罚有效，而是因为极端表现本身就是随机波动的产物
- **两次投掷的硬币**：第一次投得好的人第二次大多做得不好，第一次没投好的人第二次大多有进步——纯粹的数学现象
- **高尔顿的发现**：子代高度向平均值回归，且回归程度与母本偏离平均值的幅度成比例
- **回归无处不在**：
  - 高尔夫锦标赛：第一天表现好的选手第二天通常会稍差
  - 抑郁儿童：无论是否接受治疗，极端抑郁者一段时间后都会有所缓解
  - 聪明女性嫁"不如她们聪明"的男人：夫妻智商相关性不完美导致的数学必然
- **因果解释的诱惑**：回归效应虽可解释现象，却无法找出原因，但大脑会自动激活因果解释

## Mental Models

- **运气的作用**：极端表现 = 高于平均水平的天赋 + 好运气（或低于平均的天赋 + 坏运气）
- **保守预测原则**：对选手第二天的最佳预测应最接近平均值，而非基于第一天的分数
- **相关与回归的统一**：相关性和回归性并非两个概念，只是从不同视角对同一概念的阐释

## Anti-patterns

- **将随机波动归因于因果**：飞行教练把不可避免的随机波动与因果解释联系起来
- **错误因果解释**："今年的生意比去年好，因为去年太糟了"——虽正确描述了回归，却赋予虚假因果
- **忽视对照组**：声称某种治疗有效时，未考虑对照组仅通过回归就能改善病情
- **结果偏见**：用结果好坏评估决策质量，而非判断过程的合理性

## Key Takeaways

1. 回归平均值是数学必然，只要两个数值之间的相关度不高，就会出现回归
2. 原始数据越极端，期待的回归就越明显，因为极好的分数常常表明运气成分很大
3. 生活给予我们的反馈常常违背常理：我们因对人友好而受到惩罚（对方下次回归平均），因举止无礼而得到嘉奖（对方下次改善）
4. 要证明某种治疗方法有效，必须比较实验组和对照组——对照组仅通过回归就能改善病情
5. 理解回归概念困难的原因：系统1要求因果解释，而回归只是统计规律，没有因果故事
