# 第28章 公平性——经济交易的参照点

## Core Idea

损失厌恶是心理学对行为经济学最重要的贡献。它不仅解释了人们对得失的不对称反应，还揭示了公平性规范在经济交易中的核心作用。当前的工资、价格或租金设定了参照点，公司利用市场力量将损失强加给顾客或工人被视为不公平。公平性原则与损失厌恶相结合，解释了为什么改革难以推行、谈判难以达成、以及人们对价格变动的反应具有不对称性。

## Frameworks Introduced

- **公平性规范（Fairness Norms）**：经济交易中的道德原则，当前的工资、价格或租金设定了不可违背的参照点。
- **损失厌恶系数（Loss Aversion Coefficient）**：规避损失的动机强于获得利益的动机，比例约为2:1。
- **消极主导（Negativity Dominance）**：负面情况在众多方面战胜正面情况，坏消息比好消息更具影响力。
- **目标即参照点（Goals as Reference Points）**：未实现的目标是损失，超过目标是所得，但对未实现目标的损失厌恶更为强烈。

## Key Concepts

- **小脑扁桃核与威胁检测**：大脑对威胁性图片（如受惊吓的眼睛）的反应速度快于意识感知，威胁优先于机遇。
- **蟑螂与樱桃效应**：一只蟑螂可使一碗樱桃不再诱人，而一个樱桃对一碗蟑螂不起作用——负面压倒正面。
- **婚姻互动比例**：约翰·葛特蒙发现稳健关系需要良好互动与不好互动的比例至少为5:1。
- **计程车司机的日收入目标**：下雨天客人多、收入高时早收工，晴天客人少时反而工作更久——违背经济逻辑但符合损失厌恶。
- **高尔夫球手的标准杆效应**：职业球员击标准杆（避免损失）比击小鸟球（追求所得）更努力，成功率差异为3.6%。
- **雪铲涨价实验**：五金店在大雪后将雪铲从15美元涨到20美元，82%的人认为不公平。
- **降薪实验**：生意景气时降低现有雇员工资被视为不公平（83%），但用同样低工资雇佣新员工则被视为可接受（73%）。

## Mental Models

- **权利即参照点**：雇员有权利维持当前工资，公司有权保持当前收益，但无权将损失转嫁给他人。
- **互惠原则**：一个看似令人痛心的让步需要对方作出同样痛心的让步，双方都在假装某些利益很重要。
- **防御者优势**：拥有领地的动物在受到挑战时几乎总会获胜，损失厌恶使现状偏向成为保守力量。
- **蛋糕分配逻辑**：对越来越小的蛋糕进行商讨非常困难，因为即使损失也要分配出去；蛋糕越来越大时人们显得更随和。

## Anti-patterns

- **忽视公平性对市场的影响**：标准经济理论假设经济行为只受自身利益驱使，与公平性无关。
- **将损失强加给他人**：公司利用市场力量将自身损失转嫁给顾客或工人，会招致愤怒和报复。
- **低估惩罚的代价**：违背公平原则的雇主会受到生产力下降的惩罚，标价不合理的商人会失去销售市场。

## Key Takeaways

1. 损失厌恶是心理学对行为经济学最重要的贡献，它解释了经济交易中的诸多不对称现象。
2. 公平性原则的核心：不可利用市场力量将损失强加给他人；公司有权利保护自身收益，但个人也有权利维持当前待遇。
3. 改革通常难以推行，因为损失厌恶使潜在输家比赢家更积极、更有决心地反对变革。
4. 利他惩罚（惩罚不公平行为）会激活大脑的"快乐中心"，保持社会秩序和公平原则。
5. 法律对挽回损失和补偿失去所得之间的区分，可能凭借其对个人幸福的不对称影响而合理化。
