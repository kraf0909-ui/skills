# 第19章 "知道"的错觉

## Core Idea

人类有一种根深蒂固的错觉，认为自己了解过去、能够解释过去，进而相信自己也能预测未来。这种"知道"的错觉由三种相互关联的认知偏见共同构成：后见之明（hindsight bias）使我们事后觉得一切显而易见；叙事谬误（narrative fallacy）让我们为随机事件编造因果故事；结果偏见（outcome bias）则让我们以结果好坏来评判决策质量，而非判断过程的合理性。

## Frameworks Introduced

- **叙事谬误（Narrative Fallacy）**：纳西姆·塔勒布提出的概念，指我们为过去的憾事编造牵强的因果解释并信以为真
- **后见之明偏见（Hindsight Bias）**：事件发生后，人们夸大自己此前预测该事件的可能性，即"我早就知道"效应
- **结果偏见（Outcome Bias）**：以结果的好坏作为评判决策质量的标准，而非评估判断过程的合理性
- **光环效应（Halo Effect）**：对某人某一特质的印象影响对其所有品质的评价，保持简单和连贯
- **理论诱导的盲区（Theory-Induced Blindness）**：一旦接受某个理论作为思考工具，就很难注意到其错误

## Key Concepts

- **谷歌成功故事的错觉**：一个连贯的叙事让我们觉得自己理解了谷歌成功的秘诀，但这个故事忽略了无数可能导致不同结果的未发生事件
- **后见之明的社会成本**：
  - 导致观察者以结果评判决策，而非过程
  - 使决策者被迫采取官僚做派、极不情愿冒风险
  - 给不负责任的冒险者带来不应得的回馈
- **结果偏见的法律案例**：低风险手术发生意外后，陪审团倾向于认为手术本来就存在风险，主刀医生应更清楚
- **企业成败故事的吸引力**：光环效应 + 结果偏见 = 对商业书籍中成功秘诀的盲目信任
- **执行总裁影响力的真相**：总裁特性与公司业绩的相关系数最高约0.30，意味着能力较强的总裁只领导60%较强的公司（比随机猜测高10%）

## Mental Models

- **黑天鹅与叙事**：塔勒布指出我们关注少数已发生的重大事件，而非无数未发生的事
- **运气vs技能的混淆**：每一个重要决策都有好的结果时，坏运气本来极有可能扰乱成功步伐
- **隐形墨水**：客户责备代理人没看清墙上的笔迹，却忘了这笔迹原是用隐形墨水写成，只有在事后才变得清晰

## Anti-patterns

- **事后诸葛亮**：用"知道"一词描述事前并不确定的事后判断
- **叙事谬误**：为随机事件编造连贯的因果故事，夸大技能、低估运气
- **结果偏见**：好决策产生坏结果就责备决策者；坏决策产生好结果就赞扬其远见
- **英雄崇拜**：将企业成功过度归因于领导者个人，忽视运气和回归平均值
- **理论诱导的盲区**：接受某个理论后，对其反例视而不见

## Key Takeaways

1. 我们对过去的了解比我们自认为能够了解的要少——"知道"不是唯一引起错觉的词，"直觉"和"预感"也一样
2. 人类大脑无法妥善处理没有发生的事情，因此会夸大技能的作用、低估运气的影响
3. 后见之明偏见导致观察者不是根据判断过程的合理性来评估判断的好坏，而是以结果的好坏作为判断标准
4. 运气成分越多，从成功故事中学到的就越少——谷歌创始人曾想以不到100万美元卖掉公司
5. 《基业长青》等书中的成功和失败故事常会夸大领导风格和管理措施对公司业绩的影响，因此基本上没什么用
