# 第23章 努力养成采纳外部意见的决策习惯

## Core Idea

人们在预测和规划时，本能地采用"内部意见"——关注当前项目的具体细节、自身计划和已有进展，却忽视了"外部意见"——参考类似项目的统计结果和基础比率。这种偏向导致普遍的"规划谬误"（planning fallacy）：项目普遍超时、超支。通过采纳外部意见、使用参考类别预测，可以显著改善决策质量。

## Frameworks Introduced

- **内部意见（Inside View）**：关注具体情境，从自身经历和当前计划中寻找证据进行预测
- **外部意见（Outside View）**：将注意力转移到类似案例的统计结果上，以基准预测为锚
- **规划谬误（Planning Fallacy）**：不切实际地接近理想状况、可通过参考类似案例数据得到提高的计划和预测
- **参考类别预测（Reference Class Forecasting）**：本特·弗林夫伯格提出的方法——使用相似项目的分布信息进行预测
- **非理性坚持（Irrational Persistence）**：面对不利信息仍不愿放弃已投入大量精力的项目

## Key Concepts

- **课程编写项目的教训**：团队预测2年完成，实际用了8年；40%的类似团队失败，完成者最少用7年
- **内部意见的陷阱**：
  - 外推法错误：前几章可能比其他章节简单，投入程度也可能处于最高点
  - 忽视"未知的未知数"：离婚、生病、与官员协调等意外事件
  - 认为自己不需要了解同类案例——"每一个案例都是独特的"
- **规划谬误的典型案例**：
  - 苏格兰议会大楼：预算从4000万英镑膨胀到4.31亿英镑
  - 全球铁路项目：乘客量平均高估106%，成本平均超支45%
  - 美国厨房改造：预估18658美元，实际平均花费38769美元
- **参考类别预测三步骤**：
  1. 识别对应的参考类别
  2. 获取参考类别的统计数据，作出基准预测
  3. 如有特别原因说明该项目乐观偏差更明显，则用具体信息调整基准预测

## Mental Models

- **基准预测（Baseline Prediction）**：仅知道案例所属类别、不了解具体信息时的预测——应是进一步调整的锚定
- **弹球与罐子（再次）**：如果只知道某位女士住在纽约，预测身高应以纽约女性平均身高为基准
- **沉没成本悖论**：已投入大量精力使放弃变得困难，即使理性选择是终止项目

## Anti-patterns

- **规划谬误**：不切实际地接近理想状况，忽视类似案例的统计数据
- **偏向内部意见**：了解个别案例的人很少认为有必要了解同类别的其他案例
- **集体忽略外部意见**：即使了解到外部意见，团队仍集体忽略——苍白无力的统计学信息总被舍弃
- **案例独特性的道德辩护**："每一个案例都是独特的"——用道德立场拒绝统计思维
- **非理性坚持**：面对不利信息仍继续项目，因为放弃会令人尴尬

## Key Takeaways

1. 内部意见与外部意见的差异令人诧异——人们头脑中有相应知识，却在预测时完全未应用
2. 规划谬误无处不在：个人、政府、企业的计划和预测行为中皆不鲜见
3. 采纳"外部意见"是避免规划谬误的有效方法——使用相似团队的分布信息进行预测
4. 运转良好的组织会奖励规划师提出的精准而切实可行的方案，也会处罚未能对困难作出预测的规划师
5. 人们之所以经常承担风险项目，是因为对成功率过于乐观——乐观偏差是人们愿意承担风险的重要原因
