# 第12章 科学地利用可得性启发法

## Core Idea

可得性启发法是指人们通过"实例呈现在脑中的轻松程度"来判断概率的过程。当想要估计某类事件的出现频率时，人们会从记忆中搜寻这类问题的实例——如果搜寻过程既轻松又顺畅，这些事的发生概率就会被判断为很大。然而，这种启发法必然会产生系统性错误（可得性偏见），因为实例呈现的轻松程度受多种因素影响：事件的突出性、媒体报道、亲身经历、生动程度等，而这些因素与真实频率往往并不一致。

## Frameworks Introduced

- **可得性启发法（Availability Heuristic）**：通过"实例呈现在脑中的轻松程度"来判断概率的过程。问题的替代必然会产生系统性错误。
- **可得性偏见（Availability Bias）**：由可得性启发法导致的系统性错误。实例呈现的轻松程度受多种与真实频率无关的因素影响。
- **提取轻松度vs提取数量**：诺伯特·施瓦茨的研究表明，自我评估是由事件呈现在脑海中的轻松度来衡量的，轻松地想起某件事的体验比想起事情的数量更重要。
- **顺畅性（Fluency）**：事件在脑中呈现的轻松程度和顺畅性，是影响判断的关键因素。

## Key Concepts

- **可得性启发法的定义**：从记忆中搜寻实例，如果搜寻过程既轻松又顺畅，这些事的发生概率就会被判断为很大。
- **影响可得性的因素**：
  - 突出事件：好莱坞明星的离婚事件和政客的性丑闻格外引人注目，容易被夸大频率
  - 大事件的暂时效应：飞机失事会有媒体报道，暂时改变你对飞行安全的看法
  - 亲身经历与生动性：亲身经历、生动的图片和鲜活的例子比文字或统计数据更容易让人回想起来
- **夫妻贡献率实验**：
  - 问夫妻双方各自为保持整洁作了多大贡献（用百分比表示）
  - 自我估测的贡献率合计超过了100%
  - 可得性偏见解释：夫妻二人记自己的努力比记对方的清楚得多
- **施瓦茨的果断性实验**：
  - 列举6个果断行事的例子后，认为自己很果断
  - 列举12个果断行事的例子后，反而认为自己不够果断
  - 原因：列举12个例子很费劲，提取的轻松度下降，导致自我评估下降
  - 列举12个不果断的例子后，反而认为自己非常果断——因为想不出那么多不果断的事
- **微笑与皱眉对提取的影响**：
  - 被要求皱眉的人（认知紧张）在提取果断行为时会有难度
  - 因此会估计自己缺少果断性
- **可得性偏见的悖论**：
  - 在回忆起自己多次骑自行车的经历后，依然相信他们自己不经常骑自行车
  - 因为回忆的轻松度（而非数量）才是判断的依据

## Mental Models

- **记忆搜寻作为概率判断**：人们希望估测某一范畴的大小或某一事件的频率，但实际上却在判断自己想到相关实例的轻松程度。
- **轻松度作为信号**：系统1用提取轻松度作为频率的信号，但这个信号是嘈杂的——受突出性、媒体、情感等因素干扰。
- **可得性作为自我评估基础**：自我评估是由事件呈现在脑海中的轻松度来衡量的，而非实际数量。
- **认知紧张降低可得性**：皱眉（认知紧张）会降低提取实例的轻松度，从而影响自我评估。

## Anti-patterns

- **夸大突出事件的频率**：好莱坞离婚事件和政客性丑闻因为格外引人注目，其频率容易被夸大。
- **媒体驱动的概率扭曲**：媒体报道偏向新鲜和尖锐的事，导致人们对事件频率的估测偏离实际。
- **以亲身经历替代统计数据**：一个与你相关的判决错误会逐渐削弱你对司法体系的信任度，其影响程度比你在报纸上读到类似事件的影响更深。
- **忽视可得性偏见的高昂代价**：时刻对偏见保持警惕是件累人的事——但由此便可避免一个代价高昂的错误。
- **团队中的功劳分配失衡**：很多合作团队成员感觉他们做的事超出了自己的分内工作——可得性偏见使每个人都更容易记起自己的贡献。

## Key Takeaways

1. 可得性启发法是指人们通过"实例呈现在脑中的轻松程度"来判断概率的过程。
2. 问题的替代必然会产生系统性错误——你会发现启发法是如何通过一个简单的过程导致偏见的。
3. 夫妻二人自我估测的贡献率合计超过了100%——可得性偏见使每个人都更容易记起自己的贡献。
4. 列举12个果断行事的例子后，人们反而认为自己不够果断——提取轻松度比数量更重要。
5. 亲身经历、生动的图片和鲜活的例子比发生在别人身上的事、单纯的文字或是统计数据更容易让人回想起来。
6. 意识到自己的偏见有利于团队关系融洽——当你看到团队里的每个成员都可能感觉付出超出100%时，就足以缓和心理失衡。
7. 你必须通过自问一些问题努力重新审视自己的印象和直觉，时刻对偏见保持警惕。
