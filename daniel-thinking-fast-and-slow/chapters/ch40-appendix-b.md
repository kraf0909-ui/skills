# 附录B 选择、价值以及框架

## Core Idea

本文是卡尼曼于1983年在美国心理协会大会上的发言稿，系统总结了前景理论的核心内容。文章分析了风险性选择和无风险决策中的认知和心理物理学因素，包括S形价值函数、决策权重的非线性、框架效应、心理账户以及损失和成本的区分。这些发现表明，人类决策系统性地偏离理性选择的不变性准则，且这种偏离具有深层的心理学基础。

## Frameworks Introduced

- **前景理论（Prospect Theory）**：描述人们在风险决策中实际行为的理论，以财富变化而非最终财富状态来评估结果。
- **S形价值函数（S-shaped Value Function）**：获得时为凹面（风险规避），损失时为凸面（风险追求），损失曲线更陡（损失厌恶）。
- **决策权重的非线性（Nonlinear Decision Weights）**：极端概率被过度加权，中间概率被低估。
- **心理账户（Mental Accounting）**：人们建立心理账户来说明某个选项的优点和缺点，与多属性的参考状态相关联。
- **公式化效应（Formulation Effect）**：对结果的评估易受问题框架方式的影响。

## Key Concepts

### 风险性选择

- **伯努利的效用理论**：用收益的凹函数解释风险规避，但忽略了人们从收益、损失和中立结果角度考虑问题。
- **S形价值函数**：获得时为凹面（风险规避），损失时为凸面（风险追求），损失曲线更陡（损失厌恶系数约2:1）。
- **四重模式**：
  - 高收益高概率：风险规避（确定效应）
  - 高收益低概率：风险追求（可能性效应，如彩票）
  - 高损失高概率：风险追求（确定效应）
  - 高损失低概率：风险规避（可能性效应，如保险）
- **不变性的违背**：亚洲病害问题中，"拯救生命"框架下风险规避，"生命丧失"框架下风险追求。
- **虚假确定性效应**：人们忽略问题的第一阶段（不管是否决策，结果相同），只专注于进入第二阶段后会发生什么。

### 概率的心理物理学

- **决策权重函数**：从零到5%的增长比从30%到35%有更大影响力；从95%到100%的增长影响最大。
- **种类—边界效应**：从不可能到可能，或从可能到确定的变化比变化范围内的变化有更大影响力。
- **概率保险**：将保险费减半、只在奇数日赔付的地震保险没有吸引力——因为风险降低一半的决策权重影响很小。

### 交易与贸易

- **局部账户（Topical Accounts）**：人们自然而然地用局部账户来构架决策，使得为节省5美元去另一家店买计算器的意愿与计算器的价格相关，与夹克的价格无关。
- **禀赋效应**：人们不愿意分隔属于自己的资产，买入价格远低于出售价格。
- **损失和成本的区分**：同样的钱在第一个问题中被构架为无法补偿的损失，在第二个问题中被构架为预防损失的花费——相对于消费而言，损失更让人厌恶。
- **废弃—损失效用**：交了网球俱乐部会费后即使生病也坚持去练习，因为停止练习会迫将会费视为完全损失。

### 决策价值与体验价值

- **决策价值 vs 体验价值**：决策价值是预期结果对选项的吸引力；体验价值是结果真实体验的满足感或痛苦。
- **享乐跑步机**：快速的适应性使得任何客观进步的影响都很短暂。
- **预测困难**：享乐体验的复杂与微妙使决策制定者很难预测结果的真实体验。

## Mental Models

- **参照点即现状**：所有评估都相对于当前状态或参考状态进行，而非绝对财富水平。
- **损失厌恶的保守力量**：损失厌恶偏爱机构或个人对当前生活的细小改变，是维持稳定的引力。
- **框架即现实**：人们的偏向是关于被架构的问题，而非潜在的客观现实。
- **心理账户的层级**：最小账户（只考虑差异）、局部账户（涉及主题）、综合账户（全面考虑）。

## Anti-patterns

- **假设不变性成立**：理性选择要求偏好不随描述方式改变，但人类系统性地违背这一原则。
- **忽视框架的影响**：认为框架只是掩饰或曲解了潜在偏向的干预，而非偏向本身。
- **用单一账户评估所有决策**：人们本应使用综合账户，但 naturally 使用局部账户。
- **混淆损失与成本**：将本可视为成本的花费视为无法补偿的损失，导致非理性决策。

## Key Takeaways

1. 前景理论用财富变化而非最终财富状态来评估结果，S形价值函数解释了收益风险规避和损失风险追求。
2. 决策权重的非线性导致对极端概率的过度反应（彩票和保险的吸引力）。
3. 框架效应系统性地违背不变性原则，即使在经验丰富的受试者中也无法消除。
4. 心理账户中的局部账户导致金钱交换率的大幅变动，与标准消费者行为理论背道而驰。
5. 决策价值和体验价值的不匹配引发了许多决策问题中的额外不确定因素。
