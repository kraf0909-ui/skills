# 速查表

## 决策规则

### 判断某项事务该由哪级政府负责
| 条件 | 决策层级 |
|------|----------|
| 只影响本地，无外部性 | 地方政府全权处理 |
| 影响跨市但省内 | 省级政府协调 |
| 影响跨省 | 中央政府协调 |
| 国防、外交、国家安全 | 中央事权 |
| 社会治安、市政交通 | 地方事权 |
| 义务教育、跨省基建 | 中央地方共同事权 |

### 分析地方政府行为
| 观察指标 | 含义 |
|----------|------|
| 土地出让金占财政收入比 | 土地财政依赖度 |
| 城投债发行规模 | 隐性债务扩张速度 |
| 预算支出/收入比 | 财政自给率 |
| GDP增速目标 vs 实际 | 数据真实性 |
| 环保处罚力度 | 环保优先级 |

### 评估债务风险
| 指标 | 安全阈值 | 警示阈值 |
|------|----------|----------|
| 债务/GDP | <200% | >250% |
| 居民债务/可支配收入 | <100% | >150% |
| 地方政府债务/综合财力 | <100% | >150% |
| 短债/长债比 | <30% | >50% |

## 决策树

### 如何理解一个经济现象
```
是否涉及政府行为？
├── 是 → 查看财政资金流向
│         ├── 预算内 → 分析税收和支出结构
│         └── 预算外 → 关注土地出让、城投债
└── 否 → 分析市场机制
          ├── 价格信号是否正常 → 要素是否自由流动
          └── 是否存在垄断/管制 → 寻找制度原因
```

### 如何评判一项政策
```
政策目标明确？
├── 否 → 警惕模糊措辞，关注实际执行
└── 是 → 工具与目标匹配？
          ├── 否 → 政策可能无效或适得其反
          └── 是 → 考虑激励相容性
                    ├── 激励相容 → 政策可能有效
                    └── 激励冲突 → 执行中会变形
```

## 权衡矩阵

### 地方政府发展模式
| 维度 | 重土地、轻人 | 重规模、重扩张 | 重投资、重生产 |
|------|-------------|---------------|--------------|
| 优点 | 快速城市化 | 快速工业化 | 经济高速增长 |
| 缺点 | 房价高、公共服务不足 | 债务重、产能过剩 | 消费不足、贸易冲突 |
| 改革方向 | 以人为本、户籍改革 | 提质增效、创新驱动 | 扩大内需、改善分配 |

### 中央集权 vs 地方分权
| 维度 | 中央集权（条条） | 地方分权（块块） |
|------|-----------------|-----------------|
| 资源配置 | 统一规划 | 因地制宜 |
| 创新动力 | 较弱 | 较强（竞争驱动） |
| 协调成本 | 高 | 低（自发协调） |
| 风险 | 系统性风险集中 | 局部风险分散 |
| 适用场景 | 国防、跨省事务 | 经济发展、区域竞争 |

## 快速识别信号

### 地方政府财政压力信号
- 土地出让金增速放缓或下降
- 城投债发行利率上升
- PPP项目清理退库
- 公务员薪资拖欠传闻

### 房地产风险信号
- 房价收入比超过20倍
- 居民杠杆率快速上升
- 三四线城市库存高企
- 房企债务违约增加

### 产业结构调整信号
- 政府产业引导基金大量设立
- 特定行业补贴政策出台
- 央企重组整合加速
- 地方招商引资竞争白热化
