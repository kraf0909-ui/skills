## Core Frameworks & Mental Models

### 1. 事权划分三原则
理解中国政府间权力分配的基石：
- **外部性与规模经济**：行政区域大小应与政策影响范围一致
- **信息复杂性**：信息优势决定实际权威，下级拥有"实际权威"
- **激励相容**：让各方意愿和能力匹配，避免"上有政策，下有对策"

Use when: 分析任何政府间权力分配、央地关系、行政体制改革

### 2. 分税制改革与土地财政
1994年分税制改革深刻改变了地方政府行为：
- 地方预算支出占全国85%，但收入仅占50-55%
- 催生了"土地财政"：地方政府通过出让土地使用权获取收入
- "土地金融"将未来收入资本化，规模成倍放大
- 连接着房价、地价、财政、基础设施投资，形成复杂利益链

Use when: 分析地方政府财政、城市化融资、房地产政策

### 3. 地区间竞争模式
中国经济发展核心机制：
- "块块"式竞争（地方竞争）vs "条条"式竞争（部委竞争）
- 地方政府在招商引资、GDP增长上的激烈竞争
- 推动了快速工业化和城市化，但也造成债务、不平衡等问题

Use when: 分析区域经济发展差异、地方政府行为、产业政策效果

### 4. 政府投资与债务风险
地方政府通过城投公司深度参与经济：
- 城投公司是隐性债务的主要来源
- 债务/GDP比重10年间翻了一番（2018年达258%）
- 居民、企业、政府三部门债务互相作用
- 化解风险需要改革激励机制和财政体制

Use when: 分析地方政府债务、金融风险、基建投资

### 5. 重投资、重生产、轻消费
发展战略的结构性失衡：
- 资源向企业部门转移，居民消费占比偏低
- 国内消费不足导致产能过剩，只能对外输出
- 引发中美贸易冲突等外部摩擦
- "双循环"战略旨在纠正这一失衡

Use when: 分析经济结构、消费不足、国际贸易冲突

### 6. 生产型政府向服务型政府转型
政府角色随发展阶段转变：
- 早期：政府主导推动经济起飞和市场经济制度建设
- 现在：需要向提供公共服务的"服务型政府"转型
- 有效的市场机制本身就是不断建设的结果
- 坚持"实事求是"，区分经济发展过程和发展目标

Use when: 评判政府职能转变、改革方向、发展阶段论
